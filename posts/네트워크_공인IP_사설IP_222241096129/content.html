핫했던 게임 어몽어스에서 지원하는 Local 기능

사설 IP

​

어몽어스 Local 기능을 사용하면, 같은 공간(사설 ip 대역)에 있는 친구들끼리는 인터넷 연결이 되어있지 않아도 같은 방에 접속할 수 있었다.

​

외부에서 볼 때는 이 사설 네트워크 대역 안에 있는 사람들끼리는 다 같은 주소로 보이기 때문이다.🤫

​

공인 IP

실제 인터넷 네트워크 통신망과 통신할 때 사용하는 ip 주소.

사설IP

같은 네트워크 대역대에서 사용하는 ip 주소.

​

사설 ip를 사용하고 있는 장비들은 네트워크 통신을 할 때, 무조건 '공인 아이피로 바꿔서' 외부 세상, 인터넷과 통신한다.

​

같은 회사 안에서, 같은 학원 안에서, 같은 집에 사는 식구들끼리 집 안에서는 서로 사설 ip로 통신이 되다가, 네이버 같은 다른 네트워크 대역으로 갈 때 같은 공간에 있는 나와 친구 모두 공인 IP 하나로 바꿔져서 나가는 식.

​

지금 옆에 있는 친구, 가족, 동료들은 (같은 네트워크 대역대에서, 같은 공유기를 쓰고 있는 사람)

네이버에 내 아이피 주소 를 확인해 보자. 결과가 모두 같을 것이다. 내 핸드폰과 내 컴퓨터의 IP도 마찬가지.

​

​

네이버에서 보여주는 내 IP 주소는 네이버가 볼 수 있는 곳 까지의 IP 주소인 공인 IP 이고,

터미널에 ifconfig 를 입력해 보여지는 IP는 나의 사설 IP 이다.

사설ip는 외부에서 봤을 때 보이지 않는 ip이기 때문에, 포트포워딩 설정(4계층)을 해주지 않으면 외부에서 들어올 수 없는 공간이다.
