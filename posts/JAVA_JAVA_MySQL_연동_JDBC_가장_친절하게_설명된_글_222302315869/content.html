0. JDBC 란?

JDBC란, 자바에서 데이터베이스를 접속할 수 있도록 해주는 자바 API 이다 !

​

JDBC를 사용 이유 : 다양하게 존재하는 데이터베이스의 독립된 API를 지원 받기 위해.

​

JDBC의 목표: 애플리케이션 내에 사용 되어지는 코드로 변함 없이 다른 데이터베이스에 접근하는 것.

​

JDBC가 없다면 Oracle, MySQL 등 각각의 데이터베이스마다 제공해주는 API를 사용해야하고,

데이터베이스를 바꿔야 할 땐, 번거롭게 프로그램을 다시 만들어야 한다.

​

그치만 이 JDBC를 사용하면 어떤 데이터베이스를 쓰던 동일한 코드로 여러 종류의 데이터베이스를 접근할 수 있다.
