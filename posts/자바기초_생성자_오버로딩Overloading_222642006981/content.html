생성자 오버로딩 (Overloading)

생성자도 결국 메소드. 오버로딩 할 수 있다.

​

외부에서 제공되는 다양한 데이터들을 이용해서 객체를 초기화 하려면 생성자도 다양화될 필요가 있다.

Cat 이라는 객체 생성 시,

외부에서 제공되는 데이터가 없다면 기본 생성자로 Cat 객체를 생성해야 하고,

외부에서 name, color 데이터가 제공될 경우, Cat 객체를 생성할 수 있어야 한다.

다시말해 두 가지 경우일 때 모두 객체를 생성할 수 있어야 한다.

​

따라서 생성자가 하나뿐이라면, 두 가지 경우에 따라 객체를 모두 생성할 수 없다.

그렇기 때문에!

자바는 다양한 방법으로 객체를 생성할 수 있도록 생성자 오버로딩을 제공한다.

​

생성자 오버로딩 : 매개 변수를 달리하는 생성자를 여러 개 선언이 가능하도록 하는 것.

pubilc class 클래스 { // 생성자1 클래스 ( [타입 매개변수1, 타입 매개변수2, . . . ] ) { . . . } 클래스 ( [타입 매개변수1, 타입 매개변수2, . . . ] ) { // 생성자2 . . . } }

​

[Cat 클래스에서의 생성자 오버로딩 예시]

public class Cat { Car() { . . . } Car(String name) { . . . } Car(String name, String color) { . . . } Car(String name, String color, int old) { . . . } }

​

생성자 오버로딩시 주의할 점.

매개 변수의 타입과 개수, 선언된 순서가 똑같을 경우

매개 변수 이름만 바꾸는 것은 생성자 오버로딩으로 볼 수 없다. 구분 불가하기 때문이다.

​

Car(String model, String color) { . . . }

Car(String color, String model) { . . . } -> 이것은 오버로딩이 아니다!

​

생성자가 오버로딩 되어 있는 경우

new 연산자로 생성자를 호출할 때, 제공되는 매개변수값의 타입과 개수에 의해 호출될 생성자가 결정된다.

​

Cat cat1 = new Cat(); -> 기본 생성자로 객체 생성 Cat cat2 = new Cat("치즈"); -> Car(String name) 생성자 호출 Cat cat3 = new Cat("치즈", "노랑색"); -> Car(String name, String color) 생성자 Cat cat4 = new Cat("치즈", "노랑색", 3); -> Car(String name, String color, int old) 생성자

ex) 다양한 방법으로 Cat 객체가 생성된다.

​

[ Cat.java 생성자 오버로딩 ]

public class Cat { //필드 String name; String color; int old; //생성자 오버로딩 Car(){ } Car(String name) { this.name = name; // 필드 model 에 매개변수 값 저장 } Car(String name, String color) { this.name = name; this.color = color; } Car(String name, String color, int old) { this.name = name; this.color = color; this.old = old; } }

​

[ CatExample.java 객체 생성 시 생성자 선택 예시 ]

public class CatExample { public static void main(String[] args) { Cat car1 = new Car(); // 생성자 선택1 : 기본생성자 System.out.println(); Cat cat2 = new Car("자가용"); // 생성자 선택2 System.out.println("car2.name : "+ car2.name); System.out.println(); Cat cat3 = new Car("자가용", "빨강"); // 생성자 선택3 System.out.println("car3.name : "+ car2.name); System.out.println("car3.color : "+ car2.color); // 빨강 System.out.println(); Cat car4 = new Car("택시", "검정", 200); // 생성자 선택4 System.out.println("car4.name : "+ car4.name); System.out.println("car4.color : "+ car4.color); System.out.println("car4.maxSpeed : "+ car4.maxSpeed); } }
