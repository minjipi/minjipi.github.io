interface Ticket { ** 유효한 티켓인지/아닌지 판단하기 위해, 티켓 넘버 반환 가능하게. public int getTicketNum(); 넘버가 0인지- 0보다 작냐로 판단. 크면 유효하고 작으면 문제 있는 티켓이라 판단하게. }
class NormalTicket implements Ticket { ** 티켓 시리얼 넘버 갖고 있게. private int serial_num; public NormalTicket(int num) { this.serial_num = num; } @Override public int getTicketNum() { return this.serial_num; } }
class ErrorTicket implements Ticket { @Override public int getTicketNum() { // TODO Auto-generated method stub return 0; } ** 에러 티켓은 시리얼 번호 부여받을 필요가 없다. 무조건 0이기 때문! }
