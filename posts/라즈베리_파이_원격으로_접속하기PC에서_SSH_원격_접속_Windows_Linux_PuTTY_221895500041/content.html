1. 라즈베리 파이 ip 주소 확인

PC나 노트북에선 해당 기기의 ip를 확인할 때 ipconfig 라는 명령어를 사용하는데,

리눅스(라즈베리 파이)에서는 위 사진과 같이 ifconfig 라는 명령어를 사용해 해당 ip를 확인한다.

​

확인된 ip 주소!

​

eth에 표시된 내용은 유선 랜에 대한 네트워크 정보고, wlan에 표시된 내용은 무선 랜 정보인데,

여기서 가장 첫번째 줄의 HWaddr은 MAC 주소(랜 카드 고유번호)이고,

두번째 줄 inet addr의 값이(하얀색 마크업 된 부분)지금 우리가 쓸 IP주소가 되겠다.

2-2. PC에서 SSH 원격 접속(Windows):

Windows 에서 제공하는 원격 데스크톱 을 사용해서 라즈베리 파이에 접속하려면

라즈베리 파이에

realvnc-vnc-server를 제거 후 xrdp라는 패키지와 tightvnserver 패키지를 설치해야 한다.

sudo apt-get purge realvnc-vnc-server

위 명령어로 'realvnc-vnc-server'패키지를 제거 후,

​

sudo apt-get install tightvncserver

위 명령어로 tightvnserver 패키지를 설치한다.

​

sudo apt-get install xrdp

xrdp 패키지 또한 설치!

​

여기에 ip 주소 입력. 127.0.0.1 은 아닙니다!!!

user name - pi
