표준 입력 input() : 키보드로 입력받음 -> 표준 출력 print() : 모니터로 출력

파일에서 데이터 읽기

read(): 파일 내용 전체를 읽어들인다. (모든 내용을 하나의 문자열로 처리)

readline(): 파일 내용을 한 줄씩 읽어들인다.

inFp = None inStr = "" ​ inFp = open("data1.txt", "r", encoding="utf-8") ​ while True: inStr = inFp.readline() if inStr =="": break print(inStr, end='') inFp.close()

readlines(): 파일 내용 전체를 읽어 들인다. (리스트 형식으로 한줄한줄 읽음!)

​

​

f = open('d:\\test\\test.txt', 'rt') data = f.read() print(data) f.close()

위의 모드 중 가장 기본형인 'rt : 읽고 쓰기'

​

f = open('d:\\test\\test.txt', 'rt') datas = f.readlines() for data in datas: print(data) f.close()
f = open('d:\\test\\test.txt', 'rb')

데이터 출력

모드 w 는 파일이 존재하더라도 이전 내용에 작성하려는 내용을 덮어 쓴다.

모드 a 는 파일이 존재하면 이전 내용 마지막 라인에 작성하려는 내용을 추가한다.

​

#모드 W

f = open('d:\\test\\test2.txt', 'w') # 없는 파일에 쓰는 것. f.write("하하하하") # 쓰고픈 내용 추가 f.close()

이렇게 하고나서

f = open('d:\\test\\test2.txt', 'wt', encoding = 'utf-8') f.write("크크") # 쓰고픈 내용 추가 f.close()

w로 하게되면 덮어쓰기가 된다.

결과 :

크크 (하하하하 지워짐.)

​

#모드 a

f = open('d:\\test\\test2.txt', 'at') f.write("하하하하") # 쓰고픈 내용 추가 f.close()
f = open('d:\\test\\test2.txt', 'at', encoding = 'utf-8') f.write("크크") # 쓰고픈 내용 추가 f.close()
하하하하 크크

파일 복사 프로그램

test.txt 파일에 뭔가 쓰면

test1.txt 파일에 복사되는 프로그램 만들기

inFp, outFp = None, None inStr = "" inFp = open('c:\\test\\test.txt', "r+", encoding="utf-8") outFp = open('c:\\test\\test1.txt', "w+", encoding="utf-8") inStr = inFp.readlines() for i in inStr: outFp.writelines(i) inFp.close() outFp.close() print("-- success --")
