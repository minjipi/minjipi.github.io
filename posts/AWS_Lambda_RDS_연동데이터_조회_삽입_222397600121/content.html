3.

람다는 '서버리스'다.

람다는 컴퓨터가 아니라 pip 같은 프로그램을 쓸 수 없고, 라이브러리를 쓸 수 없는데.

​

우리가 다른 곳에서 다운받아 직접 업로드 시켜줘야한다.

이 때 zip 파일만 업로드가 가능하므로

우선 아래와 같은 방법으로 파이참에서 pymysql 다운로드 받은 후

​

해당 파일을 ZIP으로 묶어준 후 아래 아이콘으로 업로드한다.

​

업로드 하면 요렇게 파일이 보인다.

여기에 파이썬 파일을 추가하고, Deploy 후 실행시켜보자.

​

import pymysql def lambda_handler(event, context): conn = pymysql.connect( host='DB주소', user='admin', password='qwer1234', db='minji', charset='utf8') curs = conn.cursor() sql = "SELECT * FROM student" curs.execute(sql) rows = curs.fetchall() print(rows) return 0
