1. tmux의 주요 개념

session, window, pane

​

Session: tmux의 가장 큰 실행 단위, 윈도우에서는 데스크톱과 유사

Window: tmux 터미널 화면, 세션 내에서 탭처럼 이동 가능

Pane: 하나의 윈도우 내에서 창을 분할하여 여러 작업 동시에 수행 가능

​

​

tmux 명령어 정리

2.1 Session 관련 명령어

tmux: 새로운 세션 생성. 0,1,2 순서로 생성된다.

tmux ls: 세션 목록 확인. tmux 으로 새로운 세션을 만들고 tmux ls 로 확인하면 하단에 0번째 window 임이 표시된다.

tmux new -s (session_name): 세션 이름 설정

ctrl + b, $: 세션 이름 수정

ctrl + b, d: 세션 닫기(detach)

tmux attach -t (session_name): Detached 세션 다시 열기

​

​

2.2 Window 관련 명령어

ctrl + b, c: 새로운 윈도우 생성

ctrl + b, n: 다음 윈도우로 이동

ctrl + b, p: 이전 윈도우로 이동

ctrl + b, w: 모든 세션의 hierarchy를 tree 구조로 보기

​

​

2.3 Pane 관련 명령어

ctrl + b, %: 화면 세로 분할

ctrl + b, ": 화면 가로 분할

ctrl + b, z: 현재 Pane 확대 및 축소

ctrl + b, :: Pane size 조절

ctrl + b, (spacebar): Pane layout 변경

​

​

2.4 tmux configure 수정

/.zshrc 파일을 수정하여 shell 커스터마이징을 할 수 있듯

tmuz 도 ~/.tmux.conf 파일을 수정하여 커스터마이징이 가능하다.

​

ex) ctrl + b 단축키를 ctrl + s 로 커스텀

set-option -g prefix C-s kunbind-key C-b bind-key C-s send-prefix

​

ex) Mouse mode - 마우스 사용

set -g mouse on

​

그리고 내가 가장 좋아하는 동시 입력 기능

tmux에서는 현재 열린 모든 pane에 입력을 전달하는 기능을 제공한다.

여러 위치에서 동일한 작업을 수행해야 할 때 특히 유용하다.

​

​

기능 활성화:

<Ctrl-B>, : setw synchronize-panes on

'on'이나 'off'를 지정하지 않으면 현재 설정과 반대로 토글된다.

​

해당 기능도 단축키 커스터마이징이 가능하다.

vi ~/tmux.conf bind-key y set-window-option synchronize-panes

이렇게 하면 Ctrl-B + y로 간편하게 사용 가능하다.
