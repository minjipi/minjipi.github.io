1) 서비스 설치

[1] controller

dnf --enablerepo=centos-openstack-victoria,powertools -y install openstack-dashboard

​

2) 서비스 설정

[1] controller

vi /etc/openstack-dashboard/local_settings

아래를 참고하여, 위 파일 수정 및 내용 추가.

​

(1) 39번 라인 쪽에

ALLOWED_HOSTS = ['*']

​

94~99번 라인 쪽에

CACHES = { 'default': { 'BACKEND': 'django.core.cache.backends.memcached.MemcachedCache', 'LOCATION': '컨트롤러IP:11211', } }

​

105번 라인 쪽에

SESSION_ENGINE = 'django.contrib.sessions.backends.cache'

​

118번 라인 쪽에

OPENSTACK_HOST = "컨트롤러 IP"

​

119번 라인 쪽에

OPENSTACK_KEYSTONE_URL = "http://컨트롤러 IP:5000/v3"﻿

​

123번 라인쪽에

TIME_ZONE = "Asia/Seoul"

​

마지막 라인에 추가

WEBROOT = '/dashboard/' LOGIN_URL = '/dashboard/auth/login/' LOGOUT_URL = '/dashboard/auth/logout/' LOGIN_REDIRECT_URL = '/dashboard/'

저장 후 파일 나가기.

​

vi /etc/httpd/conf.d/openstack-dashboard.conf

4번 라인 쪽에 아래 내용 추가

WSGIApplicationGroup %{GLOBAL}

참고!

저장 후 나가기.

​

systemctl restart httpd.service memcached.service

​

3) 확인

웹브라우저로 http://[컨트롤러 IP]/dashboard 접속해서 확인 !
