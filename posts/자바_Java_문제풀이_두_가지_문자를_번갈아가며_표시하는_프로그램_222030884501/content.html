읽은 값의 개수만큼 *와 +를 번갈아 가며 표시하는 프로그램 작성.

​

public class Random1 { public static void main(String[] args) { // TODO Auto-generated method stub Scanner stdIn = new Scanner(System.in); System.out.println("몇개의 *+를 표시할까요?"); int n; n = stdIn.nextInt(); if (n > 0) { int i = 0; while (i < n) { if (i % 2 == 0) System.out.print("*"); else System.out.print("+"); i++; } System.out.println(); } } }

변수 i의 값을 0,1,2, ...로 증가시키는 과정에서 i가 짝수면 *, 홀수면 +를 출력해서 서로 교대로 표시되게 했다.

그런데 이 프로그램에는 두 가지 결점이 있음!

​

1) 반복할 때마다 if문을 판정한다.

while문에 의해 반복이 이루어질 때마다 if문이 실행되는데

그래서 i가 짝수인지 홀수인지 판단하는게 n번 이루어진다.

​

2) 변경에 대응하기 어렵다.

만약 시작값인 i가 1이라면,

아래와 같이 코드가 바뀐다.

{ int i = 1; while (i <= n) { if (i % 2 == 1) System.out.print("*"); else System.out.print("+"); i++; }

시작값이 바뀔 때마다 변경되는 코드는 좋다고 할 수 없다.

그래서 이 두 가지 문제를 보완한 코드를 재작성.

​

public class Random2 { public static void main(String[] args) { // TODO Auto-generated method stub Scanner stdIn = new Scanner(System.in); System.out.println("몇개의 *+를 표시할까요?"); int n; n = stdIn.nextInt(); if (n > 0) { int i = 0; while (i < n / 2) { System.out.print("*+"); i++; } if (n % 2 == 1) System.out.print("*"); } System.out.println(); } }

n이 홀수 일 때 *를 출력하고, 아닐 땐 *+를 출력한다.

예를들어 5가 들어오면, 4까지는 *+를 출력하고

마지막 5를 보고 *를 출력한다.

​

첫번째와 결과는 같지만 더 효육적인 프로그램!
