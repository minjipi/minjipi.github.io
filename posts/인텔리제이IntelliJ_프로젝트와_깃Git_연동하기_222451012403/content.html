인텔리제이(IntelliJ) 프로젝트와 깃(Git) 연동하기

GitHub에 새로운 remote repository를 만들고 기존 IntelliJ 프로젝트(모듈)을 연동하는 방법에 대해 다룬다.

과정은 다음 순서로 진행된다.

​

1) IntelliJ에 GitHub 계정 연동

2) Git에서 새 레포지토리 만들기 (GitHub Remote Repository 생성)

3) 리모트 레포지토리 등록하기

4) 로컬 레포지토리 만들기

5) commit, push

​

1. IntelliJ에 GitHub 계정 연동

깃 계정 연동

맥에서의 Preferences, 윈도우에서 Settings 를 클릭한다.

​

좌측의 GitHub 탭을 클릭하고,

계정 정보를 입력 후 [Log In]을 클릭한다.

​

버튼을 눌러 깃에 로그인 혹은 회원가입 한다.

그리고 다시 인텔리제이로 돌아와 GitHub 탭에서

​

등록된 것을 확인한다.

​

​

2. Git에서 새 레포지토리 만들기

Git은 로컬 repository가 있어서 원격 repository가 반드시 필요한건 아니지만 Git 사용자들은 대부분 원격 repository를 사용한다. GitHub에서 무료 원격 repository를 제공하므로 이를 이용하도록 하겠다.

​

GitHub 계정이 없으면 생성하고 https://github.com/new로 접속해서 GitHub repository를 생성한다.

​

Public 체크하고,

[Create repository]를 클릭하여 새 레포지토리를 생성한다.

​

그리고 생성한 레포지토리 url을 복사해둔다.

​

3. 리모트 레포지토리 등록하기

Git탭의 Manage Remotes.. 클릭

​

복사해둔 url을 넣고 OK.

​

OK

​

4. 로컬 레포지토리 만들기

VCS탭의 Create Git Repository 클릭.

​

생성한 새 프로젝트를 선택하고 Open 클릭.

​

5. commit, push

Git 탭의 Commit 클릭.

​

커밋메시지를 아무거나 써두고 Commit.

(커밋 메시지를 작성하지 않으면 반려된다. )

​

Git 탭에서 Push.

​

Push.

​

이제 깃의 레포지토리 페이지로 가서 정상적으로 Push 되었는지 확인한다.

​

이렇게 GitHub repository에 기존 모듈이 정상적으로 연동되었다.
