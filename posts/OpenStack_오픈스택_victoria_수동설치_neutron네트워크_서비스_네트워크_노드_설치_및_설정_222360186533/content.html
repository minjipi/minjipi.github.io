neutron(네트워크 서비스)

네트워크 노드 설치,설정

​

1) 서비스 설치

📡

[3] network 노드

dnf -y install centos-release-openstack-victoria sed -i -e "s/enabled=1/enabled=0/g" /etc/yum.repos.d/CentOS-OpenStack-victoria.repo dnf --enablerepo=centos-openstack-victoria -y upgrade dnf config-manager --set-enabled powertools dnf --enablerepo=centos-openstack-victoria,powertools -y install openstack-neutron openstack-neutron-ml2 openstack-neutron-openvswitch libibverbs ebtables

​

2) 서비스 설정

[3] network

mv /etc/neutron/neutron.conf /etc/neutron/neutron.conf.org <-- 백업 vi /etc/neutron/neutron.conf <-- 새로 생성
[DEFAULT] core_plugin = ml2 service_plugins = router auth_strategy = keystone state_path = /var/lib/neutron allow_overlapping_ips = True transport_url = rabbit://openstack:qwer1234@컨트롤러 IP [keystone_authtoken] www_authenticate_uri = http://컨트롤러 IP:5000 auth_url = http://컨트롤러 IP:5000 memcached_servers = 컨트롤러 IP:11211 auth_type = password project_domain_name = default user_domain_name = default project_name = service username = neutron password = 비번 [oslo_concurrency] lock_path = /var/lib/neutron/lock
chmod 640 /etc/neutron/neutron.conf chgrp neutron /etc/neutron/neutron.conf

​

(1)

vi /etc/neutron/l3_agent.ini

[DEFAULT] 섹션에 다음을 추가.

interface_driver = openvswitch

​

(2)

​

vi /etc/neutron/dhcp_agent.ini

[DEFAULT] 섹션에 다음 추가

interface_driver = openvswitch dhcp_driver = neutron.agent.linux.dhcp.Dnsmasq enable_isolated_metadata = true

​

(3)

vi /etc/neutron/metadata_agent.ini

[DEFAULT] 섹션에 다음 추가

nova_metadata_host = 컨트롤러 IP metadata_proxy_shared_secret = 비번

​

219번 라인 쪽에

memcache_servers = 컨트롤러 IP:11211

​

(4)

vi /etc/neutron/plugins/ml2/ml2_conf.ini

마지막 줄에 추가

[ml2] type_drivers = flat,vlan,gre,vxlan tenant_network_types = mechanism_drivers = openvswitch extension_drivers = port_security

​

(5)

vi /etc/neutron/plugins/ml2/openvswitch_agent.ini

마지막 줄에 추가

[securitygroup] firewall_driver = openvswitch enable_security_group = true enable_ipset = true

​

ln -s /etc/neutron/plugins/ml2/ml2_conf.ini /etc/neutron/plugin.ini
systemctl enable openvswitch systemctl restart openvswitch
ovs-vsctl add-br br-int
systemctl enable neutron-dhcp-agent neutron-l3-agent neutron-metadata-agent neutron-openvswitch-agent systemctl restart neutron-dhcp-agent neutron-l3-agent neutron-metadata-agent neutron-openvswitch-agent
