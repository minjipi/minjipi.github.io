2. 오픈스택 서비스 설치

1) keystone (인증 서비스)

(1) DB 생성

[1] controller

mysql -u root -p

CREATE DATABASE keystone;

​

GRANT ALL PRIVILEGES ON keystone.* TO 'keystone'@'localhost' IDENTIFIED BY 'DB암호';

GRANT ALL PRIVILEGES ON keystone.* TO 'keystone'@'%' IDENTIFIED BY 'DB암호';

​

(2) 서비스 설치

[1] controller

아파치서버가 파이썬 코드를 실행시킬 수 없어서, 추가 플러그인 mod_wsgi 설치.

(아파치 서버에서 파이썬 코드를 실행시킬 수 있는 플러그인)

dnf -y install openstack-keystone httpd python3-mod_wsgi

​

(3) 서비스 설정

[1] controller

vi /etc/keystone/keystone.conf

여기서.... [database] 찾아서 그 섹션 한 줄 아래에 추가

connection = mysql+pymysql://keystone:암호@controller/keystone

[token] 섹션 (2471번재 줄)) 아래에 한 줄 아래에 추가

provider = fernet

저장.

​

아래 명령어 입력.

su -s /bin/sh -c "keystone-manage db_sync" keystone

(명령어 : 아래와 같이 디비에 테이블이 생성됌.)

keystone-manage fernet_setup --keystone-user keystone --keystone-group keystone keystone-manage credential_setup --keystone-user keystone --keystone-group keystone
keystone-manage bootstrap --bootstrap-password DB암호 \ --bootstrap-admin-url http://controller:5000/v3/ \ --bootstrap-internal-url http://controller:5000/v3/ \ --bootstrap-public-url http://controller:5000/v3/ \ --bootstrap-region-id RegionOne

​

vi /etc/httpd/conf/httpd.conf

98번 라인 주석 해제 후 다음과 같이 설정

-> 변경

ln -s /usr/share/keystone/wsgi-keystone.conf /etc/httpd/conf.d/

​

systemctl enable httpd.service

systemctl restart httpd.service

​

(4) 확인

vi ~/keystone_admin

export OS_USERNAME=admin export OS_PASSWORD=qwer1234 export OS_PROJECT_NAME=admin export OS_USER_DOMAIN_NAME=Default export OS_PROJECT_DOMAIN_NAME=Default export OS_AUTH_URL=http://controller:5000/v3 export OS_IDENTITY_API_VERSION=3

source ~/keystone_admin

(환경변수 적용)
