Annotation

@ : annotation

클래스 또는 메서드 또는 변수 앞에 @Override 같은 @ 표시를 많이 봤을 것이다.

​

어노테이션이란 본래 주석이란 뜻으로, 인터페이스를 기반으로 한 문법이다. 주석과는 그 역할이 다르지만 주석처럼 코드에 달아 클래스에 특별한 의미를 부여하거나 기능을 주입할 수 있다. 또 해석되는 시점을 정할 수도 있다.

​

표준 어노테이션 (Annotation)

@Override

컴파일러에게 오버라이딩하는 메서드라는 것을 알린다.

@Deprecated

앞으로 사용하지 않을 것을 권장하는 대상에 붙인다.

@Target*

어노테이션이 적용 가능한 대상을 지정하는데 사용한다

: 어노테이션에 사용되는 어노테이션(....)

= 메타 어노테이션

= 내가 만드는 어노테이션에 쓰는 어노테이션 !

@Retention*

어노테이션이 유지되는 범위를 지정한다.

​

1. @Target

어노테이션이 적용할 위치를 선택한다.

종류는 다음과 같다.

ElementType.PACKAGE : 패키지 선언

ElementType.TYPE : 타입 선언

ElementType.ANNOTATION_TYPE : 어노테이션 타입 선언

ElementType.CONSTRUCTOR : 생성자 선언

ElementType.FIELD : 멤버 변수 선언

ElementType.LOCAL_VARIABLE : 지역 변수 선언

ElementType.METHOD : 메서드 선언

ElementType.PARAMETER : 전달인자 선언

ElementType.TYPE_PARAMETER : 전달인자 타입 선언

ElementType.TYPE_USE : 타입 선언

​

2. @ Retention

자바 컴파일러가 어노테이션을 다루는 방법을 기술하며, 특정 시점까지 영향을 미치는지를 결정한다.

어노테이션을 만들 때 꼭 있어야 함!

​

종류는 다음과 같다.

RetentionPolicy.SOURCE : 컴파일 전까지만 유효. (컴파일 이후에는 사라짐)

RetentionPolicy.CLASS : 컴파일러가 클래스를 참조할 때까지 유효.

RetentionPolicy.RUNTIME : 컴파일 이후에도 JVM에 의해 계속 참조가 가능. (리플렉션 사용)

​

커스텀 어노테이션 정의

@interface 어노테이션이름 { 타입 요소이름(); -> 애너테이션의 요소를 선언한다. }
import java.lang.annotation.Retention; import java.lang.annotation.RetentionPolicy; @Retention(RetentionPolicy.RUNTIME) public @interface Anno { public int data() default 10; 어노테이션을 사용할 때, 따로 값을 지정해 주지 않으면 default로 기본값을 지정해 줄 수 있다. }

​

커스텀 어노테이션 사용법

import anno.Anno; public class UseAnno { 지금은 여기서 5로 지정해 줬지만, 지정하지 않으면 디폴트인 10이 된다. @Anno(data = 5) public void hello() { System.out.println("hello! "); } }

​

어노테이션 테스트

import anno.Anno; import java.lang.reflect.Method; public class AnnoTest { public static void main(String[] args) { try { UseAnno useanno = new UseAnno(); Method method = useanno.getClass().getDeclaredMethod("hello"); Anno anno = method.getAnnotation(Anno.class); if (anno != null) { for (int i = 0; i < anno.data(); i++) { useanno.hello(); } } } catch (NoSuchMethodException e) { e.printStackTrace(); } } }

​

UseAnno useanno = new UseAnno();

Method method = useanno.getClass().getDeclaredMethod("hello");

: UseAnno 에 있는 "hello"라는 메서드를 가져온다.

​

Anno anno = method.getAnnotation(Anno.class);

: hello 메서드에 있는 Anno(어노테이션, data=5)을 가져온다.

​

if (anno != null)

for (int i = 0; i < anno.data(); i++)

useanno.hello();

: 어노테이션을 잘 가져왔으면 어노테이션에 있는 데이터만큼 반복한다.
