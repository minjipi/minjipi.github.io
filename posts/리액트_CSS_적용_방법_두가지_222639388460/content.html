1. HTML에서 클래스로 디자인이 지정된 것.

리액트에서는 styled 컴포넌트로 만들어서 디자인 적용한다.

​

ex)

HTML 에서 이렇게 하는 것을:

<div class="ex01"> </div>

​

React 에서는 이렇게.

const Ex01 = styled.div` 디자인; `;

​

​

2. first-child, last-child, after, before 등 디자인 적용 방법

ex) HTML

<div class="ex01"> <div class="ex02"> </div> </div>

ex02의 CSS:

.ex01:first-child { margin: 0 1.5rem 2rem; }

​

React 에서:

ex01의 자식에게 주는 디자인이기 때문에, ex01에 만들어야 함.

const Ex01 = styled.div` 디자인; &:first-child { margin: 0 1.5rem 2rem; } `;

​

​

3. 자식 클래스에게 styled component로 디자인을 적용하는 방법

ex) HTML

<div class="ex01"> <div class="ex02"> </div> </div> ex02의 CSS .ex01>.ex02 { margin: 0 1.5rem 2rem; }

​

React:

const Ex02 = styled.div` 디자인; `; const Ex01 = styled.div` <- ex01의 자식에게 주는 디자인이기 때문에 ex01에 만들어야 함 디자인; > ${Ex02} { <-CSS에서 자식과 자손이 있음, > 있으면 자식, 없으면 자손, 있으면 똑같이 잇게 해주고 없으면 없게 해줘야 함 margin: 0 1.5rem 2rem; } `;

​

​

​

4. 자식 클래스에게 className으로 디자인을 적용하는 방법

ex) HTML

<div class="ex01"> <div class="ex02"> </div> </div> ex02의 CSS .ex01>.ex02 { margin: 0 1.5rem 2rem; }

​

React

<Ex01 className='추가클래스이름'> </Ex01> const Ex01 = styled.div` <- ex01의 자식에게 주는 디자인이기 때문에 ex01에 만들어야 함 디자인; &.추가클래스이름 { <- 띄워쓰기 없이 해야 함 margin: 0 1.5rem 2rem; } `;
