0.1 Python 환경 설정(IDE) : 텍스트 에디터

비쥬얼 코드, 아나콘다, 파이참

Python 환경은 Python 코드를 실행할 컨텍스트이며 전역, 가상 및 conda 환경을 포함한다.

환경은 인터프리터, 라이브러리 및 설치된 패키지 집합으로 구성되는데

이러한 모든 구성 요소에 따라, 어떤 언어로 구성되고 구문이 유효한지, 어떤 운영 체제 기능에 액세스할 수 있으며 어떤 패키지를 사용할 수 있는지가 결정되는 것이다.

​

한 줄씩이 아닌 여러 줄의 코드를 짜려면 에디터 모드를 사용한다.
