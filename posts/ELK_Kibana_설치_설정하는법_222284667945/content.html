Kibana 설치

​

# 방화벽 끄기

systemctl stop firewalld systemctl disable firewalld setenforce 0

​

1) 키바나 설치

rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
vi /etc/yum.repos.d/kibana.repo

아래와 같이 입력.

[kibana-7.x] name=Kibana repository for 7.x packages baseurl=https://artifacts.elastic.co/packages/7.x/yum gpgcheck=1 gpgkey=https://artifacts.elastic.co/GPG-KEY-elasticsearch enabled=1 autorefresh=1 type=rpm-md

​

yum install --enablerepo=kibana kibana

​

​

2) 설정

vi /etc/kibana/kibana.yml
server.port, server.host 주석 해제 host 0.0.0.0 elasticsearch.host: 는 엘라스틱 서치 아이피로

​

3) 서비스 시작

systemctl restart kibana

​

4) 확인

웹 브라우저로 http://[키바나IP]:5601 으로 접속

​

​

에러 발생 시 로그 파일을 확인!

cat /var/log/elasticsearch/elasticsearch.log | grep "WARN" cat /var/log/kibana/kibana.log

-> 키바나 No living connections 에러
