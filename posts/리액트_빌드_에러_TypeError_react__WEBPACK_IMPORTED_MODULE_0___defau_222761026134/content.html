useEffect import의 문제였다.

​

​

에러:

TypeError: react__WEBPACK_IMPORTED_MODULE_0___default.a.createClass is not a function

​

상황:

yarn start로 로컬에서 개발하며 테스트 했을 때는 전혀 문제가 없다가 (배포시 오류도 없고, 잘 빌드되었다고 나왔는데) 접속해보면 빈 화면이 나왔다.

​

​

해결:

useEffect import의 문제였다.

수정 전 import React, { useState, useEffect } from "react"; import { Link } from "react-router-dom"; import { useEffect } from "react/cjs/react.development"; import styled, { css } from "styled-components"; import jwt_decode from "jwt-decode";
수정 후 import React, { useState, useEffect } from "react"; import { Link } from "react-router-dom"; import styled, { css } from "styled-components"; import jwt_decode from "jwt-decode";

vscode에서 자동 완성해주는 import 기능으로 useState를 선언한 것이 문제였다.

useState가 react로부터가 아닌 'react/cjs/react.development'에서 임포트 되어있던 것..

모든 파일에서 react 관련 import경로를 'react'로 바꿔주고 다시 배포하니 문제가 해결됐다.
