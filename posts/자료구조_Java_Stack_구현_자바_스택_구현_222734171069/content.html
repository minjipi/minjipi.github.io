Stack (스택)

한쪽 끝에서만 자료를 넣고 뺄 수 있는 자료구조 스택.

- 데이터를 일시적으로 보관하는 자료구조

- 선입후출 (FILO: First In, Last Out )

- 스택에 데이터를 넣는 push,

데이터를 꺼내는 pop,

push&pop이 이루어지는 꼭대기(스택 맨위의 값)를 top,

top 값을 확인하는 peek,

top의 반대인 가장 아랫부분을 bottom 이라 한다.

​

자바 스택 구현 예제

스택 생성 시, 스택크기를 결정하는 고정 길이 스택 구현

[ 기본구조 ]

  • stk : 푸시된 데이터를 저장하는 스택 배열. 인덱스 0인 요소 stk[0] 가 스택의 바닥.

  • capacity : 스택의 크기를 나타내는 int형 필드.

  • ptr : 스택에 쌓여있는 데이터 수를 나타내는 필드. 스택이 비어있을 때는 ptr=0, 가득차면 capacity 값과 같음.

  • IntStack 생성자 : 스택용 배열 본체를 생성하는 등의 준비작업. (ptr=0 하는 등..)

public class Stack { private int[] stk; private int capacity; private int ptr; // 예외처리: 스택이 비어있음. public class EmptyStackException extends RuntimeException{ public EmptyStackException(){} } // 예외처리: 스택이 가득 참. public class OverflowStackException extends RuntimeException{ public OverflowStackException(){} } // 생성자 public Stack(int maxlen){ ptr = 0; capacity = maxlen; try { stk = new int[capacity]; //스택 배열 생성 } catch (OutOfMemoryError e){ capacity =0; } } }

​

[ push 메소드 ]

public int push(int x) throws OverflowStackException { if (ptr >= capacity) //스택이 가득 찼을 때. throw new OverflowStackException(); return stk[ptr++] = x; }

스택이 다 차서 더 이상 푸시하지 못할때 OverflowStackException을 던짐.

전달받은 값 x를 stk[ptr]에 저장하고, ptr 1증가.

푸시한 값을 반환.

​

[ pop ]

public int pop() throws EmptyStackException { if (ptr <= 0) //스택이 비어있음. throw new EmptyStackException(); return stk[--ptr]; }

스택의 꼭대기에 있는 값을 제거하고 그 값을 반환.

스택이 비어서 꼭대기 값을 pop하지 못할 때 EmptyStackException 던짐.

ptr 값을 1 감소시키고 그 후 stk[ptr]에 저장된 값을 반환.

​

[ peek ]

꼭대기 값을 '들여다보는' 메소드.

public int peek() throws EmptyStackException { if (ptr <= 0) throw new EmptyStackException(); return stk[ptr - 1]; }

​

[ clear ]

스택의 모든 요소를 한번에 삭제

public void clear() { ptr =0; }

스택에서 푸시, 팝하는 등 모든 작업은 ptr을 바탕으로 이루어짐.

모든 요소를 삭제하려면 ptr을 0으로 하면 된다.

​

[ indexOf ]

public int indexOf(int x) { for (int i= ptr-1; i>=0; i--) //꼭대기부터 선형 검색 if (stk[i]==x) return i; // 검색 성공 return -1; // 검색 실패 }

stk에 값의 데이터(x)가 포함되어 있는지, 배열 몇번째에 있는지를 알려주는 메소드.

꼭대기부터 선형검색을 하는데, 이는 먼저 pop되는 데이터를 찾기 위해서.

값을 찾으면 그 값의 인덱스를 반환하고, 실패하면 -1을 반환.

​

[ getCapacity ]

스택 용량 확인. capacity를 반환.

​

[ size ]

스택에 쌓인 데이터 수(ptr값)을 반환.

​

[ isEmpty ]

스택이 비어있으면 true, 아니면 false 반환.

​

[ isFull ]

스택이 가득 찼으면 true, 아니면 false 반환.

​

[ dump ]

스택 데이터 모두 출력.

public int getCapacity() { return capacity; } public int size() { return ptr; } public boolean isEmpty() { return ptr <= 0; } public boolean isFull() { return ptr >= capacity; // ptr == capacity; 로도 할 수 있음 } public void dump() { if (ptr <= 0) System.out.println("스택이 비어있음!"); else { for (int i = 0; i < ptr; i++) System.out.print(stk[i] + " "); System.out.println(); } }

"있음." : "있지않음.")); System.out.println("가득 " + (s.isFull() ? "차 있음." : "차 있지않음.")); break;
