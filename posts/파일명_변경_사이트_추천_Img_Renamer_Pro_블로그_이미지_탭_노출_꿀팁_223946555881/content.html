Intro.

블로그 포스팅을 하다 보면 수십 개의 이미지 파일명을

한번에 변경해야 하는 경우가 굉장히 자주 생깁니다.

​

참고로 이렇게 바꿔주는 이유는

포스팅에 첨부하는 이미지 파일명이나 설명도

네이버 키워드로 잡히기 때문이에요.

네이버 블로그 좀 해보신 분들은 이미 아시겠죠?ㅎㅎ

​

하나하나 파일명 바꾸는게 귀찮아서

안하신 분들이 많을 텐데요.

저도 귀찮아서 프로그램을 만들어서 쓰다가

이런 꿀팁은 공유하는게 좋겠다 싶어서

웹사이트로 만들어 봤습니다.

​

File Renamer

메타 이미지 랩스가 제공하는 File Renamer 서비스. 파일명 일괄 변경을 100% 로컬에서 안전하게 처리하세요. 간단 업로드로 수백 개 파일명을 일괄 변경하고, ZIP 다운로드·개별 저장을 지원합니다. 개인정보 유출 없는 브라우저 내 처리로 안심하고 파일명 일괄 변경을 수행하세요.

iiimaze.github.io

​

사용법은

1) 이미지 드래그 앤 드롭으로 슥 옮겨서,

2) 바꾸고 싶은 파일명 입력,

3) 앤터만 누르면 끝입니다.

​

참 쉽죠?!

​

네이버 이미지 탭에 노출시키시고 싶으신 이미지는

꼭 설정 해보시는게 좋아요!

​

​

특히 이 사이트가 좋은 점은

진짜 안전한 사이트라는 점입니다.

보통의 파일명 변경기 프로그램, 사이트들은

서버에 파일을 업로드해서 처리하는 방식인데요.

​

이러면 사이트 만든 사람 서버에

내가 업로드한 사진들이 다 저장되거든요.

​

이 방법은 100% 보안상 우려가 있고,

대용량 파일 처리 시 시간도 오래 걸립니다.

​

그래서 저는 완전히 클라이언트 사이드에서 동작하는

이미지 파일명 일괄 변경기를 만들어보았습니다.

​

​

주요 기능

  • 일괄 파일명 변경: 선택한 접두어로 모든 파일명을 접두어_001.jpg 형태로 변경

  • 다양한 포맷 지원: JPG, PNG, GIF, WebP는 물론 iPhone의 HEIC 파일까지 지원

  • 이미지 파일 포함, 모든 파일 지원.

  • 실시간 미리보기: 변경될 파일명을 미리 확인 가능 (HEIC빼고 ㅎ)

  • 드래그 앤 드롭: 파일을 드래그해서 간편하게 추가

  • ZIP 다운로드: 변경된 파일들을 ZIP으로 압축해서 다운로드

  • 실시간 파일명 미리보기 & 진행률 표시는 센스 ^^
