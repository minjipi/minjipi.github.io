th:classappend : 클래스 동적 추가

<div th:with="testClass = ${code == '01' ? : 'active' : ''}"> <li class="comm" th:classappend="${testClass}"> 코드가 01일때만 클래스 추가 </li> </div>

not #strings.isEmpty( data ) : 문자열 null 체크, 빈값 체크

<th:block th:with="testText = ${not #strings.isEmpty(info.title) ? info.title : '입력해주'}"> </th:block>

th:selected : select박스 selected

<select name="selectPhone"> <option th:selected="${phone}=='010'">010</option> <option th:selected="${phone}=='011'">011</option> <option th:selected="${phone}=='012'">012</option> </select>

th:if, th:unless

<div th:if="${data.type == '01'}"> 타입이 01일때 태그 노출 </div> <div th:unless="${data.type == '01'}"> 타입이 01이 아닐때 태그 노출 </div>
