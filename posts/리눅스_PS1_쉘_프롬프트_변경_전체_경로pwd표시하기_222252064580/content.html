쉘 변수, PS1

현재 사용자의 시스템 사용환경을 개별적으로 사용할 수 있도록 지원하는 사용자별 시스템 환경변수를 쉘 변수라 한다. 쉘변들 가운데 PS1 이라는 쉘 변수가 있는데 이 쉘 변수는 우리가 사용하는 쉘 변수의 모양을 결정한다. 쉘 변수들을 확인하려면 echo $SP1 명령어를 실행해 보면된다.

​

현재 사용자의 쉘변수 확인하기

출력 명령어 echo를 사용해 $PS1를 확인.

[root@localhos]# echo $PS1 [\u@\h \W]\$

​

:: $PS1 변경 (재부팅 시 저장되지 않음.)

[root@localhost]# PS1="[\u@ \W]\\$ "

​

전체 사용자에게 적용 (재부팅 후에도 유지)

vi /etc/bashrc

/etc/bashrc : 모든 사용자들에게 적용되는 쉘 사용환경 설정파일

vi 혹은 vim 편집기로 bashrc 파일을 열어서 아래 구문:

[ "$PS1" = "\\s-\\v\\\$ " ] && PS1="[\u@\h \w]\\$ "﻿

PS1="[\u@\h \w]\\$ " 여기서 쌍따옴표 안 부분을 마음대로 수정하면 된다.

​

쉘 프롬프트에 전체 경로 표시

쉘 프롬프트에 pwd 표시.

아래와 같이 \W 부분을 삭제하고, $PWD 입력하고,

양쪽을 ""(큰따옴표) 가 아닌 ''(작은 따옴표)로 둘러싼다.

PS1='[\u@]h $PWD ]\$ '

​

​

개별사용자 적용 (나만 적용)

vi ~/.bashrc

.bashrc : 모든 사용자의 홈디렉토리에 존재하며, 이 파일에 각 개별 사용자들의 쉘 사용 환경을 설정할 수 있다.

​

홈디렉토리에서 .bashrc 파일을 연후, PS1변수변경내용을 적어주면 됨.

vi /home/사용자/.bashrc ...(생략) PS1="[\u@\h \w]\\$ "

위 구문처럼 PS1="[\u@\h \w]\\$ " 를 맨아래에 추가해 준다.

PS1 변수에 사용되는 기호들과 의미

\t 현재시간(HH:MM:SS)24시간

\T 현재시간(HH:MM:SS)12시간

\@ 현재시간(HH:MM 오전/오후)12시간

\d 현재날짜(요일 월 일)

\s 현재 사용중인 쉘의 이름(C쉘=/bin/csh, bahs쉘=/bin/bash)

\w 현재 디렉토리의 전체경로

\W 현재디렉토리명

\u 사용자명

\h 서버의 호스트명

\H 서버의 도메인명

\# 접속한 순간부터 사용한 명령어의 번호를 1부터 차례대로 표시, 명령어 사용횟수

\! history 수

\\$ 현재 사용자가 root(UID=0)면 #, 아니면 $표시

\\ “\”

\a ASCII 종소리 문자(07)

\e ASCII의 escape문자(033)

\n 개행문자(줄바꾸기)

\v 사용중인 bash버전

\V 사용중인 bash의 배포, 버전+패치수준

\r Carrage return

\nnn 8진수 nnn에 해당하는 문자
