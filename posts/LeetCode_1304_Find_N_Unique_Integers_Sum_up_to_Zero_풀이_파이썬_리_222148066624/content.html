1304. Find N Unique Integers Sum up to Zero

​

문제:

Given an integer n, return any array containing n unique integers such that they add up to 0.

예시 1:

Input: n = 5 Output: [-7,-1,1,3,4] Explanation: 다음과 같은 해답도 정답 -> [-5,-1,1,2,3] , [-3,-1,2,-2,4].

​

예시 2:

Input: n = 3 Output: [-1,0,1]

아하~

더해서 0이 나오는

n 길이의 배열을 구하는 것! 이군요

​

n = 3 -> 이건 대충 테스트 하기위해 n을 3으로 해보았습니다 output = [] # n이 홀수일 경우 if n % 2 == 1: output.append(0) # n이 짝수일 경우 for i in range(int(n / 2)): i += 1 output.append(i) output.append(-1 * i) print(output)
