1. useState

useState는 이렇게 생겼다.

const[count, setCount]

기본적으로 Getter와 Setter다.

Getter는 변수의 값을 가져오는데 사용하고, Setter는 변수 값을 변경하는데 사용한다.

정확히는 useState()가 호출되면 배열을 반환하는데, 그 배열의 첫번째 원소는 상태값이고 두번째 원소는 상태를 업데이트하는 함수다.

이 함수에 파라미터를 넣어서 호출하게 되면 전달받은 파라미터로 값이 바뀌게 되고 컴포넌트는 정상적으로 리렌더링 된다.

​

그래서 이 코드는 변수 word를 만들고

const [word, setWord] = useState("");

word의 상태값을 변경할 수 있는 함수 setWord 를 만든 것이다.

그리고 일단 useState 값을 "", 즉 아무것도 없게 해뒀다.

​

2. onChange

<Input type="text" onChange={(e) => setWord(e.target.value)}></Input>

input 태그의 타입을 text로 주고,

onChange 함수(값이 변할때 발생하는 리스너)를 사용해

변한 값 e 로 word를 세팅한다.
