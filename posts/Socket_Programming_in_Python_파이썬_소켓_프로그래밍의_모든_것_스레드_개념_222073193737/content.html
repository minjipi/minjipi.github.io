1. 소켓 만들기_ 클라이언트 편

우선 간단하게 소켓 생성에 대해 알아보자.

아래는 소켓(클라이언트) 생성 기본 코드이다.

sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM) # 서버에 접속하기 위한 소켓 생성 sock.connect(('192.168.1.100', 12345))

server에 접속하기 위한 client socket을 생성,

주소와 포트번호를 설정한다.

message=input() sock.send(message.encode("UTF-8")) # 서버에 메시지 전달 data = sock.recv(65535) print(data.decode("UTF-8"))

그리고 connect()함수를 이용하여 서버에 접속을 시도하고, send()함수를 이용해 메시지를 전달한다.

여기까지 클라이언트 코드가 완성 되었다!

import socket sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM) sock.connect(('192.168.1.198', 12345)) message = input() sock.send(message.encode("UTF-8")) data = sock.recv(65535) print(data.decode("UTF-8"))

여기서 이 코드를 실행시키면

[ 대상 컴퓨터에서 연결을 거부했으므로 연결하지 못했습니다.] 라는 에러메시지가 뜬다.

대상 컴퓨터 : 서버가 아직 켜지지 않아서 연결할 수가 없는 상황인 것이다.

12345 포트 열려있지 않는 것을 확인했다.

2.

#2 서버의 IP와 포트를 연결! server_socket.bind(('0.0.0.0', 12345)) ## 서버의 IP와 포트를 연결

우선 소켓을 설정하고, bind()함수를 통해 주소 정보를 할당한다.

위 코드는 IP 0.0.0.0, 즉 모든 아이피를 받으면서,

.bind(("'', 12345)) 포트를 사용하는 데이터를 받는다는 의미이다.

​

또 .bind(("'', 80))은 시스템에 있는 모든 주소로 소켓에 연결할 수 있음을 나타낸다.

그리고 또! 낮은 번호의 포트는 일반적으로 잘 알려진 서비스(HTTP, SNMP 등)를 위해 예약되어 있다.

때문에 지금 실습에서는 적당히 높은 번호(한 4 자릿수)를 사용해야 한다.

​

3.

server_socket.listen(0) ## 서버의 포트를 LISTENING 상태로 변경

listen()함수를 통해 연결 수신 대기 상태로 전환한다.

listen에 대한 인자는 외부 연결을 거부하기 전에 최대 5개의 연결 요청을 큐에 넣기를 원한다는 것을 소켓 라이브러리에 알리고 있다.

​

4. client가 연결할 시 accpet() 함수를 이용하여 연결을 수락한다.

client_socket, addr = server_socket.accept()

이제 우리는 포트 12345 에서 대기하는 서버소켓을 가지고 있고, 웹 서버의 메인 루프를 입력할 수 있다.

이제 코드의 나머지 부분이 제대로 작성되었다면 끝이다!

​

5. 만약 client가 보낸 메시지가 있을 경우, recv(byte크기)를 이용하여 메시지를 가져온다.

data = client_socket.recv(65535)

나머지 부분 포함 전체 코드는 아래에..

server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM) server_socket.bind(('0.0.0.0', 12345)) ## 서버의 IP와 포트를 연결 server_socket.listen(0) ## 서버의 포트를 LISTENING 상태로 변경 client_socket, addr = server_socket.accept() data = client_socket.recv(65535) print(data.decode("UTF-8")) message = input() client_socket.send(message.encode("UTF-8"))

이 코드를 실행 시키고 포트 번호를 확인해 보자.

스레드, Thread

프로세스에서 작업을 수행하는 단위.

일반적으로 하나의 프로그램은 하나의 스레드를 가지고 있지만

둘 이상의 스레드를 동시에 실행할 수도 있다.

파이썬에서는 함수를 만들고 함수를 스레드 모듈로 실행시키면 해당 함수는 스레드로 동작한다.

​

스레드 확인 예제를 살펴보도록 하겠다.

import threading def count(low, high, add): for i in range(low, high, add): print("count_thread %d" % i) t1 = threading.Thread(target=count, args=(1, 100, 2)) t1.start() t2 = threading.Thread(target=count, args=(2, 100, 2)) t2.start() print("main_thread")

위 코드의 결과:

이 스레드 라는 녀석을 가지고

끊기지 않는 소켓 프로그램은 만들어 보자.

import socket import threading svr_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM) svr_socket.bind(('192.168.1.28', 777)) svr_socket.listen(0) client_socket, addr = svr_socket.accept() def recv_data(): while True: data = client_socket.recv(65535) print(data.decode("UTF-8")) client_socket.close() def send_data(): while True: msg = input() client_socket.send(msg.encode("UTF-8")) client_socket.close() recv_thread = threading.Thread(target=recv_data) recv_thread.start() send_thread = threading.Thread(target=send_data) send_thread.start() while True: pass
