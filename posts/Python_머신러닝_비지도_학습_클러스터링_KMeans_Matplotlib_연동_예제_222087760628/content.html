# 3 from sklearn.cluster import KMeans import matplotlib.pyplot as plt import pandas as pd features = [[140, 1], [130, 1], [160, 0], [170, 0]] data = pd.DataFrame(features) data.columns = ['kg', 'surface'] feature = data[['kg', 'surface']] feature.head() #################: 전 포스팅에서 했던 내용 model = KMeans(n_clusters=2) model.fit(features) predict = pd.DataFrame(model.predict(feature)) predict.columns = ['predict'] r = pd.concat([feature, predict], axis=1) plt.scatter(r['kg'], r['surface'], c=r['predict'], alpha=0.5) #scatter <- 요게 표를 그려줌! # kg 가 x축 값, surface가 y축 값, predict가 투명도. plt.xlabel("kg") plt.ylabel("surface") plt.legend() plt.show() # 파이참 환경인 경우 .show를 해야함!

Iris Dataset 란?

sklearn 에서 제공하는 데이터셋인 Iris dataset 으로써,

위 데이터셋은 Iris, 붓꽃의 서로 다른 3가지 종(setosa, versicolor, virginica)의

sepal(꽃받침), petal(꽃잎) feature를 포함하고 있습니다.

데이터셋은 다음과 같은 내용을 포함하고 있습니다.

array([[5.1, 3.5, 1.4, 0.2], [4.9, 3. , 1.4, 0.2], ... [5.9, 3. , 5.1, 1.8]])

각 row는 하나의 데이터 sample 을 나타냅니다. 각 column은 feature를 나타내며

순서대로 sepal length(꽃받침의 길이), sepal width(꽃받침의 넓이), petal length(꽃잎의 길이), petal width(꽃잎의 넓이) 를 의미합니다.

이렇게 load한 데이터는 Bunch 라는 객체로 생성되며,

이 객체의 속성으로 .data 이외에 데이터의 label인 .target 과 데이터에 대한 설명인 .DESCR 등이 있는데

자세한 설명은 공식 문서를 참조하시기 바랍니다ㅎㅎ

3. k-Means 알고리즘 + Matplotlib 연동 예제

아래 코드를 입력해 주세요.

K-Means 알고리즘 순서 입니다.

# 3 from sklearn.cluster import KMeans k_means = KMeans(n_clusters=3) k_means.fit(data) k_means_predicted = k_means.predict(data) centroids = k_means.cluster_centers_ target_names = iris.target_names plt.figure(figsize=(10, 7)) ax = plt.axes(projection='3d') ax.scatter(data[k_means_predicted != target, 0], data[k_means_predicted != target, 1], data[k_means_predicted != target, 3], c='k', s=50) ax.scatter(centroids[0, 0], centroids[0, 1], centroids[0, 3], c='r', s=50, label='centroid1') ax.scatter(centroids[1, 0], centroids[1, 1], centroids[1, 3], c='g', s=50, label='centroid2') ax.scatter(centroids[2, 0], centroids[2, 1], centroids[2, 3], c='b', s=50, label='centroid3') ax.legend() plt.show() print(k_means_predicted)

​

K-Means 알고리즘 수행 과정

1. 모든 샘플들을 공간상에 뿌리고 k개(우리가 분류하고 싶은 데이터 갯수=3개)의 중심점 위치를 랜덤하게 잡습니다.

2. 그리고 k개의 중심점을 기준으로 공간을 나눈 다음,

나누어진 각 공간에 속한 점들의 평균 위치로 중심점을 이동시켜서

3. 각 데이터들을 가장 가까운 centroids가 속한 그룹에 할당합니다.

4. 2번 과정에서 할당된 결과를 바탕으로 centroids를 새롭게 지정합니다.

이런 과정을 반복하여 k개의 최종 중심점 위치를 찾아냅니다!

이렇게 다양한 결과가 도출되는 이유는,

비지도 학습에서는 사과가 0이다 라고 알려준 적이 없으니까

비지도 학습에서는 사과가 0이다 라고 알려준 적이 없기 때문에,

예를 들어서 사과를 1이라고도 답했다가 0이라고도 답했다가

실행할 때마다 바뀌는 결과가 나오는 것 입니다.

답을 뱉어주는 것이 아니라

그냥 0과 1을 '구분'만 하는 상황!

​

아이리스 데이터니까 우리가 원하는 답은 0 1 2 이런건데,

1 0 2 이렇게도 답하고

1 2 0 이렇게도 답하고 하는 상황이죠
