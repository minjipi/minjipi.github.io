# 이미지 만드는 이유 : 컨테이너 생성. 오케스트레이션을 생각!

# 전체 구성

리렉터리안에 아래 3가지 파일이 있어야 한다.

dbconn.jsp | Dockerfile | mysql-connector-java-8.0.25.jar

* jar 파일은 공식 홈페이지에서 받았다.

https://dev.mysql.com/downloads/connector/j/

​

​

# DB 준비 :: MySQL

1. MySQL 이미지 생성

docker pull mysql

​

2. MySQL 이미지 생성 확인

명령어로 다운로드한 Docker 이미지를 확인.

​

3. MySQL 이미지 실행

docker run --name mysql-container -e MYSQL_ROOT_PASSWORD=password -d mysql:latest

* 위 명령어대로 하면 한글 깨짐 현상이 있다. 한글이 깨지지 않도록 설정하려면 아래 인자값을 넣어주어야 한다.

--character-set-server=utf8mb4 --collation-server=utf8mb4_unicode_ci

* 여기서 포트 -p 3306:3306 은 해 줄 필요가 없다. 목표는 결국

DB 컨테이너로부터 값을 가져오는 'tomcat 이미지 만들기'기 때문에 외부에서 접속할 일이 없기 때문이다. 이따가 만들 톰캣서버와의 연동은 어차피 같은 네트워크 대역이기 때문에 포트 옵션을 줄 필요가 없다.

​

* -e, --env=[] 명령어로 컨테이너에 환경 변수를 설정한다. 보통 설정 값이나 비밀번호를 전달할 때 사용.

-e MYSQL_ROOT_PASSWORD=examplepassword

​

​

4. 컨테이너에서 MySQL 컨테이너 생성 확인

컨테이너 리스트 출력

docker ps -a

​

5. MySQL Docker 컨테이너 접속

docker exec 명령을 사용하여 docker 컨테이너에 접근해 MySQL에 로그인한다.

docker exec -it mysql-container bash

​

접속 후 데이터 생성.

mysql -u root -p Enter password: password ##### Welcome to the MySQL monitor. Server version: 8.0.25 MySQL Community Server CREATE DATABASE minji; use minji; CREATE TABLE minji_tbl (sname VARCHAR(10), sage INT); INSERT INTO minji_tbl VALUES('pig',10); INSERT INTO minji_tbl VALUES('bird',5); ############## CREATE USER 'minji'@'%' identified by 'password'; GRANT ALL PRIVILEGES ON minji.* TO 'minji'@'%'; FLUSH PRIVILEGES; # 확인 use mysql; select user, host from user;

# Dockerfile

FROM tomcat:latest # 기본이미지는 tomcat 이미지를 쓰겠다. 근데 여기는 mysql-connector 같은게 안깔려 있으니까 추가로 깔겠다 RUN mkdir /usr/local/tomcat/webapps/minji # 톰캣 프로젝트 폴더에 sjb 폴더를 만들겠다. ADD ./mysql-connector-java-8.0.25.jar /usr/local/tomcat/lib/ # 호스트PC에 있는 jar 파일을 /usr/local/tomcat/lib/ 여기에 추가하겠다. ADD ./dbconn.jsp /usr/local/tomcat/webapps/minji/dbconn.jsp # 호스트PC에 있는 dbconn.jsp 파일을 /usr/local/tomcat/webapps/sjb/dbconn.jsp 여기에 추가하겠다. EXPOSE 8080 # 외부에서 8080 포트로 접속할 수 있게 하겠다. CMD ["/usr/local/tomcat/bin/catalina.sh", "run"] # 이 이미지로 컨테이너를 생성 시, 마지막에 이 명령어를 실행하겠다.

​

​

dbconn.jsp

집어넣을 파일

<%@page import="java.sql.*"%> <%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%> <!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd"> <html> <head> <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"> <title>Insert title here</title> </head> <body> <% Connection conn = null; ResultSet rs = null; String dbip = System.getenv("DBIP"); String dbname = System.getenv("DBNAME"); String dbuser = System.getenv("DBUSER"); String dbpass = System.getenv("DBPASS"); String url = "jdbc:mysql://"+dbip+":3306/"+dbname+"?serverTimezone=UTC"; String id = dbuser; String pwd = dbpass; try { Class.forName("com.mysql.jdbc.Driver"); conn = DriverManager.getConnection(url, id, pwd); Statement stmt = conn.createStatement(); String sql = "SELECT sname FROM student"; rs = stmt.executeQuery(sql); while(rs.next()) { out.println(rs.getString("sname")); } conn.close(); } catch (Exception e) { e.printStackTrace(); } %> </body> </html>

​

* 위 코드 <3 tier architecture / 웹 서버, 웹 어플리케이션, DB 서버 연동> 때 사용했던 코드와 비교.

바뀌는 값들은 컨테이너에서 환경변수를 전달받게 코드를 작성.
