문자열이 입력되면 try-catch문을 이용해서 InputMismatchException 에

'다시 입력하시오'라는 메시지 출력 하기.

​

* 방금 for(;;) 대신 while (true) 라고 바꾸는 것이 더 예뻐 보인다는 말을 듣고 바꾸었다.!!

import java.util.InputMismatchException; import java.util.Scanner; public class Gugudan { public static void main(String[] args) { System.out.println("1부터 9까지의 임의의 수를 입력하쇼"); try { while (true) { Scanner sc = new Scanner(System.in); int ndan = sc.nextInt(); if (ndan < 10) { for (int i = 1; i < 10; i++) { System.out.println(ndan + "x" + i + "=" + (ndan * i) + ""); } } else { System.out.println("1부터 9까지 다시 입력하쇼"); continue; } } } catch (InputMismatchException e) { System.out.println("엇? 다시"); } } }

처음 코드. 근데 이렇게 하고보니..

숫자키가 아닌 문자키를 잘못 눌렀을때, 바로 에러가 나오는 것이 맘에 들지 않았다.

​

그래서, 처음엔

int ndan = sc.nextInt();

요 부분을 int 가 아닌 문자열로 바꿀까? 했다.

String ndan = sc.next();

*참고로 자바 스캐너 사용 방법은 저번 포스팅에 자세히 적어두었당 *_*vV
