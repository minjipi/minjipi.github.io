1. if~else와 elif구문

단순히 '이거면? 이것. 아니면 저것' 일 때,

조건이 2개일 때는 if와 else를 사용하면 되고..

​

3가지 이상의 여러가지 조건이 있을 때 쓰는 조건문은,

파이썬에서 elif 를 사용한다.

아래 코드로 elif 사용법을 알아보자.

>>> number = input("정수 입력> ") 정수 입력> 30 >>> number = int(number) >>> >>> if number > 0: ... print("양수네요") ... elif number < 0: ... print("음수네용") ... else number == 0: ... print("헐 0이다") ... 양수네요

이 elif를 사용해서 예제 1.1 의 코드로 조금 더 응용해 보았다.!

>>> name = '민지' >>> >>> if name == '민지': ... print('확인') ... elif name == '바보': ... print('입장불가') ... elif name == '민정': ... print('누구세요?') ... else: print('이름이 뭔가요?') #출력 이름이 뭔가요? 확인

Pass 키워드

코드를 짜다보면 일단 전체적으로 골격을 잡아놓고 차근차근 채워나가야지~ 하는 마음으로 아래와 같이 짜두는 경우가 있다.

if zero == 0 //나중에 else : //나중에

다른 언어의 경우 에러가 뜨지 않는데

파이썬에서는 if 조건문에서 들여쓰기를 하지 않으면 indentationError가 발생한다.ㅜㅜ

그래서

if zero == 0 0 else : 0

이렇게 0을 쓰기도 했다.

그치만 이건 그냥 딱 봐도 이상하고..

왜 0이지...? 라는 느낌이 들 수 밖에 없다.

파이썬에서는 이런 고민을 좀 덜어주기 위해 pass라는 키워드를 제공한다.

​

'pass = 쫌 이따 할라고 비워둠'

>>> number = input('정수> ') 정수> 5 >>> number=int(number) >>> if number > 0: ... pass ... else: ... pass

이렇게 틀만 만들어 두고 싶을 때 사용한다.
