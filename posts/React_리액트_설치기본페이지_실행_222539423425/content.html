0. node.js 설치 & 설치 확인

​

아래 링크에서 node.js를 설치한다.

https://nodejs.org/ko/download/

다운로드 | Node.js

Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.

nodejs.org

​

설치가 됐는지 아래 명령어로 확인해보자.

node -v

혹은 npm -v 로 확인한다.

​

1. cmd 창에서 cd 명령어를 이용해 생성한 폴더 경로로 이동한다.

​

2. 명령어로 yarn을 설치한다.

npm install -g yarn

* 안된다면 npx로 해도 됌.

​

잘 설치됐는지 확인한다.

yarn -v

​

3. create-react-app 설치

아래 명령어로 설치한다.

npm install -g create-react-app

​

4. create-react-app가 설치됐으면 명령어로 study(이름은 본인 마음)라는 프로젝트를 생성한다.

create-react-app study

​

* 2022.1.18 갑자기 위명령어로 생성이 안돼서

설치되어 있는 yarn 명령어로 다시 하니까 생성이 됐다..

yarn create react-app study

​

생성 후 study 경로를 보면 package.json 등의 파일과 폴더가 생성된 것을 확인할 수 있다.

​

5. react 서버 실행

방금 생성한 study 폴더에서 다음 명령어로 리액트 서버를 실행한다.

yarn start

그러면.. 아마 자동으로 웹페이지가 뜨겠지만,

크롬 웹 브라우저에서 http://localhost:3000 으로 접속해보면

기본 페이지가 뜰 것이다.
