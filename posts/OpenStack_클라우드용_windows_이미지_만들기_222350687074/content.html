4. ISO 파일로 qcow2 이미지 파일 생성

사용할 드라이버 설치

wget https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/archive-virtio/virtio-win-0.1.185-2/virtio-win-0.1.185.iso

​

가상의 하드 디스크 생성

qemu-img create -f qcow2 /virt/windows7.qcow2 20G

​

5. ISO로 가상머신 생성

virt-install --name windows7 --ram 2048 \ --disk /virt/windows7.qcow2,format=qcow2 \ --network network=default \ --graphics vnc,listen=0.0.0.0 --noautoconsole \ --os-type=windows --os-variant=win7 \ --cdrom=/virt/ko_windows_7_ultimate_k_with_sp1_x86_dvd_u_677508.iso
virt-install --name win7 --ram 2048 --vcpus 1 \ --network network=default,model=virtio \ --disk path=/virt/windows7.qcow2,format=qcow2,device=disk,bus=virtio \ --cdrom /virt/ko_windows_7_ultimate_k_with_sp1_x86_dvd_u_677508.iso \ --disk path=/virt/virtio-win-0.1.185.iso,device=cdrom \ --vnc --os-type windows --os-variant win7
