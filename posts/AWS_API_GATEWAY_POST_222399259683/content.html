1. Lambda 함수 생성

pymysql.ZIP 파일을 다운받아 업로드한다.

첨부파일
pymySQL.zip
파일 다운로드

바이러스 아님.. ㅜㅜ🐬

찾아서 받기 귀찮으신 분들 위에거 다운받으세용

​

​

코드는 다음과 같다.

import pymysql def lambda_handler(event,context): conn = pymysql.connect( host='RDS 엔드포인트', user='admin', password='비번', db='DB이름', charset='utf8') curs = conn.cursor() sql = "INSERT INTO student VALUES('"+event["sname"]+"',"+str(event["sage"])+");" curs.execute(sql) conn.commit() return "success"
