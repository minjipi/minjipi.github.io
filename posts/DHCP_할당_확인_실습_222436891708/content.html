DHCP

DHCP 통신방식 : 브로드 캐스트

후보를 알려주면 - 응답 - 할당 - 응답

​

DHCP 할당 확인 실습

yum install dhcp

​

​

10분 마다

재시작

systemctl restart dhcpd

방화벽 끄고

​

67번 포트 확인

netstat -anlp | grep :67 udp 0 0 0.0.0.0:67 0.0.0.0:* -/dhcpd udp 0 0 0.0.0.0:67 0.0.0.0:* -/dnsmasq

서버 컴퓨터에서

tail -f /var/lib/dhcpd/dhcpd.leases ... # The format of this file is documented in the dhcpd.leases(5) manual page. # This lease file was written by isc-dhcp-4.2.5

이렇게 할당 받아가는 것 실시간 확인하게 해두고

클라이언트 컴퓨터 준비.

아래 명령어로 클라이언트 컴퓨터 네트워크 설정 파일에 들어가 몇 가지 수정한다.

vi /etc/sysconfig/network-scripts/ifcfg-ens33

들어가서

static 을 dhcp 로 수정.

하고 시스템 재시작.

systemctl restart network
