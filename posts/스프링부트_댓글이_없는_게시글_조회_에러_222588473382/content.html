[ 해결 ]

게시글 조회 시 댓글이 없어도 에러가 안나게,

게시글 조회 service에 if로 예외처리를 해줬다.

public 게시글DTO readService(int idx) { Set<RcommentDTO> rcommentDTOSet = new HashSet<>(); result.forEach(arr->{ Rcomment rcomment = (Rcomment) arr[3]; if (rcomment!= null) { RcommentDTO rcommentDTO = RcommentDTO.builder() .idx(rcomment.getIdx()) .writer(rcomment.getWriter()) .contents(rcomment.getContents()) .build(); rcommentDTOSet.add(rcommentDTO); } }); }
