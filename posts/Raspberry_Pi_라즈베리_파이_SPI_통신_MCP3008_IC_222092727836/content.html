조도센서는 빛의 세기에 따라 저항값이 변하는 소자다.

그런데 라즈베리파이는 DAC(Digital-Analog Converter) 뿐 아니라

ADC(Analog-Digital Converter) 장치도 내장하고 있지 않다.

아날로그 신호를 디지털로 변환해 주는 ADC를 통해 변환된 아날로그 신호를 받고,

받은 신호를 라즈베리파이에서 읽기 위해서는

라즈베리파이에서 SPI 프로토콜을 사용할 수 있게 해주어야 한다.

SPI란?

​SPI(Serial Peripheral Interface)는 I2C,CAN, UART와 같은 시리얼 통신(직렬 통신) 방식 중 한가지로 마이크로컨트롤러, 스프트 레지스터, SD카드 등의 소형 주변 장치 사이에 데이터를 전송하기 위한 기능.

SPI 통신의 특징

→1대 다수의 통신을 지원하는 동기식 통신 방식

→동시에 송수신이 가능함

→I2C 에 비해 속도가 빠름

→하나의 마스터 장치만 지원함

이제 ADC 기능을 가진 MCP3008 IC를 이용해

아날로그에서 변환된 디지털 값을 SPI 통신을 받아오자.

​

#MCP3008IC

요렇게 생겼다

MVP3008은 8개의 ADC 채널이 있고 해상도는 10bit.

아날로그 전앖을 2^10 만큼 분해해서 디지털로 변환 가능하다는 의미.

​

코드 :

# -*- coding: utf-8 -*- import time import spidev delay =0.5 pot_channel=0 spi=spidev.SpiDev() spi.open(0,0) spi.max_speed_hz=100000 def readadc(adcnum): if adcnum > 7 or adcnum < 0: return -1 r = spi.xfer([1,8+adcnum<<4,0]) data=((r[1]&3)<<8)+r[2] return data while True: pot_value = readadc(pot_channel) print("=====") print("POT Value: %d" % pot_value) time.sleep(delay)

SPI 회로

MCP3208 SPI 통신 세부사항

1)ADC칩 MCP3208

MCP3208은 8채널 12비트 아날로그 디지털 컨버트입니다. 센서의 출력 값인 아날로그 값을 12비트의 디지털 값으로 변환하는 칩입니다. 라즈베리 파이는 ADC 기능이 없어서 센서를 이용하지 못하는 단점이 있는데 MCP3208과 같은 외부 ADC칩을 사용하여 센서를 사용할 수 있습니다. 현재는 라즈베리파이와 MCP3208 간에 SPI 통신을 이용하여 ADC 데이터를 받아와야 합니다.

SPI 통신 프로그램은 마스터와 슬레이브로 나뉩니다. 현재는 라즈베리파이가 마스터이고, MCP3208이 슬레이브입니다. 즉 마스터인 라즈베리파이가 슬레이브인 MCP3208에 정해진 명령을 보내면 그에 해당되는 값을 전송해주는 시스템입니다. 우리는 센서 값을 ADC하여 라즈베리파이가 수신할 필요가 있으므로 다음과 같은 규칙으로 전송해야 합니다. 라즈베리파이는 총 3개의 바이트를 전송해주는데, 그 내용은 다음과 같습니다.

첫째 바이트 0 0 0 0 0 Start SGL D2

둘째 바이트 D1 D0 x x x x x x

셋째 바이트 x x x x x x x x

이 데이터중 아래에서부터 12비트가 ADC 결과 값입니다.
