#a link

링크를 추가할 때 사용하는 <a> 태그.

링크를 만들 때는 2가지 정보가 필요하다.

이동하려는 실제 웹페이지 주소(=URL)와 내가 만든 웹페이지에서 보여지는 링크 이름을 입력.

보여지는 링크 이름은 URL 주소를 그대로 써도 되고, 원하는 단어를 써도 ok.

​

#pre

HTML 문서는 다른 문서 프로그램과 달리 띄어쓰기나 줄바꿈이 입력한대로 적용되지 않음.

몇 칸을 띄어쓰건, 몇 줄을 바꾸건 그냥 하나의 줄에 한 칸 띄어쓰기로 보여지기 때문에 불편함을 느낄 때가 많은데

이런 경우에 <pre> 태그를 사용할 수 있다.

​

<pre> 태그 안의 내용은 스페이스바/탭키/엔터키를 사용한 그대로 보여짐.

​

예를들어 스페이스 바를 세 번 누르면 화면에서 세 칸 띄어쓰기가 되어 보이고,

엔터 키를 한 번 누르면 바꾸면 화면에서도 줄이 바뀐다.

​

사용 방법:

입력한대로 보여주려는 텍스트를 <pre>~</pre> 사이에 써주기.

<pre> 태그는 한 줄이 보여지는 폭이 정해져 있고 자동 줄바꿈이 안된다는 것.

이럴 경우 폭을 넘어서는 내용은 다음 줄로 넘어가는 것이 아니라, 잘려서 보이게 된다..

#table

위의 표는 3개의 행으로 이루어져 있다.

첫 번째 행은 데이터를 구분하는 항목 이름인데, 이것을 헤더(header)라 부른다.

​

즉, 첫 번째 행은 이름, 나이라는 2가지 헤더로 구성되어 있고

두 번째 행부터 실제 값:

두 번째 행은 김현우, 20이라는 데이터(data)

세 번째 행은 이미영, 25라는 데이터(data)

​

표를 만들 때는

<table> ~ </table> 태그 안에 행의 개수만큼 표의 행(table row)을 뜻하는 <tr> ~ </tr>을 사용.

각 헤더 항목은 <tr> ~ </tr> 안에 표의 헤더(table header)를 뜻하는 <th> ~ </th>를 사용한다.

마찬가지로 각 데이터 항목은 <tr> ~ </tr> 안에 표의 데이터(table data)를 뜻하는 <td> ~ </td>.

​

[실습.1]

<table> <tr><th>헤더1</th><th>헤더2</th></tr> <tr><td>데이터1</td><td>데이터2</td></tr> </table>

#table-border :

테두리가 생기긴 했으니 선 색을 입혀 주자.

​

테두리를 뜻하는 속성명은 border이고 속성값은 굵기, 선스타일, 색 등이 있다.

선 굵기는 px로 표시해주고.

선 스타일에서 실선은 solid, 점선은 dotted, 이중실선은 double

선 색은 글자색과 똑같이 red, black 등의 색 이름, hex값, rgb값 중 하나로 정한다.

​

표에서 테두리는 table, header, data 모든 셀에 추가 해야 한다.

그렇지 않으면 테두리가 일부에만 나타나는 것을 볼 수 있을 것이다.

​

표 테두리를 추가하는 방법:

<head> <style> table, th, td {border : 4px solid black;} </style> </head>

<head> 태그 안에 <style> 태그를 쓰고 그 안에 관련 코드를 쓰고,

스타일 지정 범위는 <table> 태그, header를 나타내는 <th>, data를 나타내는 <td>이다.

border 다음에 콜론(:), 그 다음에 굵기, 한 칸 띄고 선 스타일, 한 칸 띄고 선 색을 쓰면 된다.

​

스타일 지정 범위만 바꿔 주면 table, th, td 대신 h1이나 p 태그를 쓰는 식으로,

표 외의 다른 요소에도 테두리를 추가할 수 있다.

​

[실습.2]

<table> <head> <style> table, th, td { border: 4px solid black; } </style> </head> <tr> <th>헤더1</th> <th>헤더2</th> </tr> <tr> <td>데이터1</td> <td>데이터2</td> </tr> </table>

위에 나타나는 이중 테두리를 없애려면 아래의 한 줄만 더 추가해 주면 된다.

border-collapse: collapse;
<table> <head> <style> table,th,td { border: 4px solid black; border-collapse: collapse; } </style> </head> <tr> <th>헤더1</th> <th>헤더2</th> </tr> <tr> <td>데이터1</td> <td>데이터2</td> </tr> </table>

#드롭다운 리스트 : Dropdown List

많은 양의 옵션 중 단 한 가지의 옵션을 골라야 할 때, radio button도 좋지만 더 많은 옵션이 필요하다면 드롭다운 리스트를 사용하는 것이 좋다.

<label for="lunch-menu">점심메뉴</label> <select id="lunch-menu" name="lunch"> <option value="pizza">피자</option> <option value="curry">카레</option> <option value="salad">샐러드</option> <option value="ramen">라면</option> <option value="tacos">타코스</option>

Dropdown list를 구현하기 위한 태그는 select 태그이다.

id 속성은 label 태그를 위함이고 name 속성은 사용자가 선택한 리스트의 value를 담기 위함이다.

select 태그 안에서 option 태그를 생성하여 드롭다운 리스트의 옵션들을 추가할 수 있다.

사용자가 피자를 점심 메뉴로 선택하고 제출했다면 "lunch = “pizza” 데이터가 넘어간다.

​

#Textarea Element :

사용자 후기, Q&A 등 웹사이트 사용자에게 장문의 글을 받아야 할 때는 textarea 태그를 사용하는 것이 좋다.

<label for="review">제품후기</label> <textarea id="review" name="review" rows="8" cols="80"></textarea>

첫 번째로 textarea 태그의 id 속성을 label 태그와 연결해준다.

그 다음 textarea의 크기를 조정해줄 row 속성과 cols 속성을 입력해준다.

입력란의 우측 하단부를 드래그해서 크기를 사용자가 직접 조정할 수도 있다.
