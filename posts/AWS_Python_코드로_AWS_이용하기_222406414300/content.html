Python 코드로 'AWS' 이용하기

https://boto3.amazonaws.com/v1/documentation/api/latest/index.html

[내 보안 자격증명] 탭에 들어간다.

​

​

만약 키가 있고 키 파일도 저장되어 있다면 상관없지만,

키가 없거나 키 파일이 저장되어 있지 않으면 새 액세스 키 만들기로 키를 만들어준다.

만들어준 다음, credentials 파일을 만들어 키파일의 내용을 넣어주자.

​

cd ~ mkdir .aws vi credentials

​

홈디렉토리에 .aws 라는 이름의 폴더를 만들고,

그 안에 credentials 파일을 생성한다.

credentials 파일의 내용은

아래 양식에 맞춰서 본인의 루트키 파일 내용을 넣으면 된다!

​

[default] aws_access_key_id = 본인키 aws_secret_access_key = 본인키 region = ap-northeast-2

​

예시:

띄어쓰기에 주의해서 파일 내용을 작성 후 저장하고 나간다.

(윈도우의 [내 문서] 혹은 맥의 홈디렉토리 경로에 한글이 있으면 오류가 나는 경우가 있다ㅜㅜ)

​

파이참 프로젝트를 하나 만들고,

아래 명령어로 boto3 을 설치한다.

pip install boto3

​

그리고 다음과 같이 코드를 작성하고, 실행.

import boto3 s3 =boto3.resource('s3') for bucket in s3.buckets.all(): print(bucket.name)

Python 코드로 '인스턴스 상태 확인'

import boto3 ec2 = boto3.resource('ec2') for status in ec2.meta.client.describe_instance_status()['InstanceStatuses']: print(status)

Python 코드로 'Cloudwatch'

https://boto3.amazonaws.com/v1/documentation/api/1.10.46/guide/cw-example-metrics.html

import boto3 cloudwatch = boto3.client('cloudwatch') paginator = cloudwatch.get_paginator('list_metrics') for response in paginator.paginate(Dimensions=[ {'Name': 'InstanceId','Value':'i-0ede24dc771fa25b5'} ], MetricName='CPUUtilization', Namespace='AWS/EC2'): print(response['Metrics'])
