RDS 접속 :

우선 mySQL 서버 프로그램이 깔린 곳에서 접속할 수 있으니, 프로그램 설치.

sudo apt-get install mysql-server-5.7

​

RDS 접속 명령어 :

mysql -u [Master username] -p -h [엔드포인트 주소]

​

RDS는 아이피 대신 엔드포인트로 접속한다.
