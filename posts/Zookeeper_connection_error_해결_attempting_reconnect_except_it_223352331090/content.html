에러 로그:

session 0x0 for server localhost/127.0.0.1:2181, closing socket connection. attempting reconnect except it is a sessionexpiredexception.

​

상황:

zkCli.sh 실행 후 작업 불가능. ls 등 작업 시 cli 환경에서 튕기는 현상.

​

​

위 에러 로그는 주키퍼 사용 시 다양한 이유로 발생한다.

나의 경우, 보안 이슈로 iptables 방화벽 설정했던 것 때문에 자기 자신과 통신이 안됐던 것으로 확인.

이 경우 zkCli.sh 실행 시 다음처럼 -server 옵션을 주면된다.

​

./zkCli.sh -server {ip}
