1. 네트워크 설정

(1) SSH enable 설정 파일 생성

SD카드 최상위 폴더에 파일이름이 ssh 인 파일을 생성, 내용 및 확장자는 없음

메모장 혹은 vi로 파일을 만들어 줍니다.

​

(2) Wifi 접속 설정 (친구컴의 랜선으로 할거면 상관없음)

SD카드 최상위 폴더에 파일이름이 wpa_supplicant.conf 파일 생성

﻿ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev update_config=1 network={ ssid= "접속할 공유기의 이름" // "" 요거 있어야 함 psk="접속할 공유기의 PW" }

(4) IP 설정

vi /etc/dhcpcd.conf

위 명령어로 접속해서

아래처럼 파일 수정해줌.

2. 초기설정

- sudo raspi-config 입력 후 설정

- Interfacing Options -> SSH -> Yes 를 선택하여 SSH를 활성화

- Interfacing Options -> VNC -> Yes 를 선택하여 VNC 서버를 활성화

- Advanced Options -> Resolution -> DMT Mode 9 를 선택하여 기본 해상도를 설정

* Default 값만 아니면 됨.

- Finish 를 선택.
