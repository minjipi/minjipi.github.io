MCP란?

MCP (Model Context Protocol)

LLM이 외부 데이터 소스와 표준화된 방식으로 연결되어,

실시간 정보에 접근할 수 있도록 해주는 통신 프로토콜입니다.

​

MCP를 한마디로 간단하게 설명하면

"AI가 내 컴퓨터의 프로그램들과

직접 소통할 수 있게 해주는 다리"

라고 할 수 있습니다.

​

기존에는 챗지피티한테

"노션에 오늘 할 일 추가해줘"라고 하면

"죄송합니다. 직접 접근할 수 없어요"라고 했는데,

이제 클로드는 진짜로 내 노션에 들어가서

할 일을 추가해주는거죠.

​

말 그대로 AI 비서가 내 컴퓨터 안에서

일을 대신해주는 시대가 온 건데요.

우선, 제일 만만하게 일 시켜볼만한

노션 MCP를 사용해봅시다.

​

* 사실 노션이 제일 만만한게 아니고

제일 깔끔하고 편리해요.🙂

​

​

노션 MCP 연동하기

0. node 설치

바닷길을 건너려면, 배가 필요한 것처럼

이번 과정에서도 꼭 필요한게 있습니다.

node 라는 건데요.

아래 명령어 한 줄로 node를 설치해줍니다.

​

# macOS (Homebrew) brew install node # Windows # https://nodejs.org 접속 → LTS 버전 다운로드 및 설치 # Ubuntu sudo apt update sudo apt install nodejs npm -y

​

1. Claude Desktop 설치

Claude Desktop 버전 어플리케이션 다운로드

https://claude.ai/download

​

그리고 본인 OS에 맞는 버전으로

Claude Desktop를 설치합니다.

​

​

2. Notion API 키 발급

아래의 Notion Developer 페이지에 접속해

로그인해주세요.

​

https://www.notion.so/profile/integrations

The AI workspace that works for you. | Notion

A tool that connects everyday work into one space. It gives you and your teams AI tools—search, writing, note-taking—inside an all-in-one, flexible workspace.

www.notion.so

​

"새 API 통합 만들기" 클릭하고,

아래와 같이 설정합니다.

​

​

새 API 통합 이름 설정하고, 유형은 프라이빗 선택.

​

그럼 '프라이빗 API 통합 시크릿'가 생성되는데,

이걸 복사해둡니다.

​

3. Notion API와 내 노션 페이지 연결

좀 전에 생성한 API와 노션 페이지를 연결해서

API가 노션 페이지에 접근 가능하게 해줍시다.

​

노션에서 새 개인 페이지를 추가하고,

새 개인 페이지를 추가

새 페이지에서 우측 상단 점 세개 클릭 > 연결 > 좀 전에 만든 API 를 선택해줍니다.

​

확인 눌러주세요.

​

​

4. Claude Desktop 설정

Claude Desktop 앱에서 아래 이미지와 같이

'설정'을 눌러주세요.

​

​

'개발자' 탭 > 설정 편집 클릭해주세요.

​

​

Claude Desktop 의 설정 파일인 claude_desktop_config.json 에

다음 내용을 추가합니다.

​

{ "mcpServers": { "notionApi": { "command": "npx", "args": ["-y", "@notionhq/notion-mcp-server"], "env": { "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer [API 통합 시크릿 붙여넣기]\", \"Notion-Version\": \"2022-06-28\" }" } } } }

​

OPENAPI_MCP_HEADERS 에

아까 복사해둔 '프라이빗 API 통합 시크릿'을

붙여넣기 해주세요.

​

그리고 Claude Desktop을 재시작 해줍니다.

Claude Desktop 프로그램을 완전히 껐다가

다시 켜주세요.

​

5. Claude Desktop 재시작

AI 비서 탄생

Claude Desktop 재시작하고,

Claude에게 일을 시켜 봅시다.

​

저는 노션 새 페이지를 만들어보고 하고,

거기에 오늘 날짜 페이지를 만들어

점메추를 부탁했어요.

​

결과가 너무 기늑해서 아래 동영상으로 찍어봤습니다 ^^
