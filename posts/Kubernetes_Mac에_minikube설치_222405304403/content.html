Mac에서 kubectl과 minikube를 셋팅하기 위해서 brew를 활용한다.

​

Error: Permission denied @ apply2files - /usr/local/lib/docker/cli-plugins

미니쿠베 설치 시 다들 나는 에러!! 해결은 간단하다.

​

에러 메시지 확인 후, chown -R 로 권한 부여.

sudo chown -R minz /usr/local/lib/docker

그리고 다시 설치!

5. calico.yaml 을 받아서 수정한다.

​

https://docs.projectcalico.org/manifests/calico.yaml

해당 사이트에서 내용을 긁어와 메모장이나 vi 편집기로

vi calico.yaml

라는 이름의 파일을 만들어 내용 붙여넣기 후 저장한다.

​

(아래와 같이 명령어로도 할 수 있다.)

dnf -y install wget wget https://docs.projectcalico.org/manifests/calico.yaml

​

위 파일에서 아래 부분을 찾는다.

보면 #주석과 띄어쓰기 한 칸이 있는데

​

주석, 띄어쓰기를 지워준다.

# - name: CALICO_IPV4POOL_CIDR # value: "192.168.0.0/16" ======== 아래와 같이 수정 ======== - name: CALICO_IPV4POOL_CIDR value: "192.168.2.0/24"

​

변경 사항 적용.

kubectl apply -f calico.yaml

​

확인:

kubectl get pods -n kube-system

조금 기다리면 calico-kube-controllers-7f4f5bf95d-2vll4
