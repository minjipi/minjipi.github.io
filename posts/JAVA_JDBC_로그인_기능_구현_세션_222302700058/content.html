🔨 세션 코드 작성

- first.html : 세션을 저장하는 링크 구현해둔 페이지

- check.jsp : session.getAttribute()로 세션을 확인하는 JSP

- set.jsp : session.setAttribute()로 세션을 저장하는 JSP

​

이 3개의 파일 생성.

​

first.html <!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Insert title here</title> </head> <body> <a href="save.jsp">세션 저장하는 곳으로 이동</a> </body> </html>
check.jsp <%@ page contentType="text/html;charset=UTF-8" language="java" %> <html> <head> <title>check</title> </head> <body> <% if (session.getAttribute("value1") != null) { out.println(session.getAttribute("value1")); } else { out.println("저장된 세션 없음."); } %> </body> </html>
save.jsp <%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%> <!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Insert title here</title> </head> <body> <% String get_session = (String)session.getAttribute("key"); if(get_session != null) { out.println("저장된 세션 값 : " + get_session); } else { out.println("세션 없음"); } %> </body> </html>
