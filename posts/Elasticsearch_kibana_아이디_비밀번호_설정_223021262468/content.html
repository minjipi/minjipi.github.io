Elasticsearch 비밀번호 설정 방법

7.x 버전 부터 비밀번호 설정 할 수 있도록 xpack 에 추가 되었다.

​

1)

vi /etc/elasticsearch/elasticsearchelasticsearch.yml

아래 추가

xpack.security.enabled: true xpack.security.transport.ssl.enabled: true

2) 비밀번호 변경

/usr/share/elasticsearch/bin/elasticsearch-setup-passwords interactive

Y 선택 후 패스워드 설정

Enternter password for [elastic]:

Reenter password for [elastic]:

Enter password for [apm_system]:

Reenter password for [apm_system]:

Enter password for [kibana_system]:

Reenter password for [kibana_system]:

Enter password for [logstash_system]:

Reenter password for [logstash_system]:

Enter password for [beats_system]:

Reenter password for [beats_system]:

Enter password for [remote_monitoring_user]:

Reenter password for [remote_monitoring_user]:

위 순서로 패스워드가 설정됨

​

3) 설정후 elasticsearch 재시작

elasicsearch {ip}:9200 접속하면 로그인 하라고 하는데

elastic / 설정한 비밀번호 입력

* ES_PATH_CONF 어쩌고는 환경변수 설정하면 된다.

키바나 비밀번호 설정

vi /etc/kibana/kibana.yml

elasticsearch.username: "elastic" elasticsearch.password: "password"

kibana 재시작 후 접속

{IP}:5601

​

접속하면 로그인 아이디 비밀번호 입력폼이 나오고, elasticsearch와 동일하게 elastic/비밀번호 입력
