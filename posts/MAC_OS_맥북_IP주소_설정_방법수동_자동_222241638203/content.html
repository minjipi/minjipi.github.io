MAC OS IP주소 설정 방법(수동, 자동)

​

1. 시스템 환경설정 > 네트워크

​

2. 네트워크 > '고급'

​

3. TCP/IP 탭 >

​

여기서 DHCP 사용으로 하면

자동으로 IP 설정하는 것.

스크롤버튼을 눌러보자.

​

​

수동 주소로 DHCP 사용을 클릭.

여기서 ip 주소를 수동으로 바꾸면 된다.

​

위에서부터 ip주소, 서브넷 마스크, 라우터.

설정을 해 주고

​

4. DNS 서버 추가

DNS 를 추가해 준다.

​

구글 DNS인 8.8.8.8 입력 후 확인 클릭.

이걸 안하면 와이파이에 연결은 되어 있지만 인터넷을 못하는 상황이..😱
