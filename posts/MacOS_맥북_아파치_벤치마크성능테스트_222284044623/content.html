✔️벤치마크(성능테스트)

특정 오브젝트에 대해 일반적으로 수많은 표준 테스트와 시도를 수행하여

오브젝트의 상대적인 성능 측정을 목적으로 컴퓨터 프로그램을 실행하는 행위.

​

​

✔️'Apache Benchmark'

AB는 커맨드 라인을 활용한 매우 가볍고 유용한 웹서버 벤치마킹 도구

간단한 REST API나 컨텐츠에 대한 성능 테스트 시에 빠르고 간편하게 벤치마킹 정보를 얻어올 수 있다.

​

웹서버가 설치되어 있는 시스템의 성능을 측정하거나 튜닝할 때나 간단하게

response time 의 측정 정보만 측정해야하는 경우 사용함.

​

​

💻맥북 Apache Benchmark

아파치 실행

apachectl -v

Apache Benchmark 설치

brew install homebrew/apache/ab
# ex ab.exe -n 4000 -c 2000 http://192.168.100.30:8080/db.jsp﻿
