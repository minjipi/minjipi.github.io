1. Opencv 설치하기

1) 필요한 패키지 및 라이브러리 설치

패키지 리스트 업데이트 및 샐치된 패키지들 업그레이드

﻿apt-get update apt-get upgrade

재부팅 명령어 : init 6

​

소스코드 컴파일을 위한 패키지 설치

﻿apt-get install build-essential cmake

​

이미지 포맷 및 비디오 코덱 설치

﻿apt-get install libjpeg-dev libtiff5-dev libjasper-dev libpng12-dev
﻿apt-get install libavcodec-dev libavformat-dev libswscale-dev libxvidcore-dev libx264-dev libxine2-dev

​

Video4Linux라는 리눅스에서

실시간 비디오 캡처 및 비디오 제어를 위한 패키지 설치

﻿apt-get install libv4l-dev v4l-utils

​

오픈 그래픽 라이브러리 설치

﻿apt-get install mesa-utils libgl1-mesa-dri libgtkgl2.0-dev libgtkglext1-dev

​

기타 라이브러리 설치

﻿apt-get install libatlas-base-dev gfortran libeigen3-dev apt-get install python2.7-dev python3-dev python-numpy python3-numpy

​

2) OpenCV 및 추가 라이브러리(contrib) 설치하기

압축을 풀어 줄 디렉토리 생성 및 이동

mkdir ~/opencv cd ~/opencv

​

소스코드 다운 및 압축 해제 (오래 걸림ㅠㅠ)

wget -O opencv.zip https://github.com/opencv/opencv/archive/4.4.0.zip unzip opencv.zip wget -O opencv_contrib.zip https://github.com/opencv/opencv_contrib/archive/4.4.0.zip unzip opencv_contrib.zip

​

SWAP 메모리 설정

vi /etc/dphys-swapfile 에서->

CONF_SWAPSIZE=100 을 다음처럼 변경

﻿CONF_SWAPSIZE=2048

​

새로 설정했으니까 재시작

/etc/init.d/dphys-swapfile restart

​

설치(컴파일 및 설치)

일반적으로는 아래와 같이 하지만!!

** 요 바로 아래껀 따라하는거 아님 **

﻿./configure make make install

우리는 이렇게 해요 !!!!

﻿cd opencv-4.4.0 mkdir build cd build

그래서

​

cmake -D CMAKE_BUILD_TYPE=RELEASE \ 아래로 쭉 옵션들 넣기 -D CMAKE_INSTALL_PREFIX=/usr/local \ -D WITH_TBB=OFF \ -D WITH_IPP=OFF \ -D WITH_1394=OFF \ -D BUILD_WITH_DEBUG_INFO=OFF \ -D BUILD_DOCS=OFF \ -D INSTALL_C_EXAMPLES=ON \ -D INSTALL_PYTHON_EXAMPLES=ON \ -D BUILD_EXAMPLES=OFF \ -D BUILD_TESTS=OFF \ -D BUILD_PERF_TESTS=OFF \ -D ENABLE_NEON=ON \ -D ENABLE_VFPV3=ON \ -D WITH_QT=OFF \ -D WITH_GTK=ON \ -D WITH_OPENGL=ON \ -D OPENCV_ENABLE_NONFREE=ON \ -D OPENCV_EXTRA_MODULES_PATH=../../opencv_contrib-4.4.0/modules \ -D WITH_V4L=ON \ -D WITH_FFMPEG=ON \ -D WITH_XINE=ON \ -D ENABLE_PRECOMPILED_HEADERS=OFF \ -D BUILD_NEW_PYTHON_SUPPORT=ON \ -D OPENCV_GENERATE_PKGCONFIG=ON ../

make -j4

make install
