1. PC와 모바일이 같은 와이파이를 잡도록 설정한다. PC와 모바일이 다른 와이파이를 잡고 있으면 로컬이 안 열림.

ip 찾는 법은 맥버전과 윈도우 버전 둘 다 써본다.

​

================================

[맥 버전 ip 찾는 법 ]

터미널을 열어서 아래 명령어로 ip 주소 확인

ifconfig | grep inet

​

[윈도우 버전 ip 찾는 법 ]

1. PC에서 와이파이창을 열어준다.

윈도우 우측 하단에서 해당 아이콘을 눌러 와이파이 창을 열어주세요. 버튼을 누르면 와이파이 연결창이 뜹니다.

2. '속성'을 클릭한다.

3. 속성 창이 열리면 'IPv4 주소'에 적혀있는 IP를 확인한다.

================================

​

3. 모바일에서 웹을 연다.(아이폰은 사파리, 안드로이드는 크롬)

그리고 주소창에 [아까 찾은 아이피] : [포트번호]

ex) 123.123.123.123:3000

​

* 포트번호는 PC에서 로컬 열었을 때 링크 보면 나와있다.

나는 리액트라 포트번호 3000 입력함.

​

4. 모바일에서도 확인 가능!
