(1) DB 생성

[1] controller

mysql -u root -p

CREATE DATABASE placement;

GRANT ALL PRIVILEGES ON placement.* TO 'placement'@'localhost' IDENTIFIED BY 'qwer1234';

GRANT ALL PRIVILEGES ON placement.* TO 'placement'@'%' IDENTIFIED BY 'qwer1234';

​

​

(2) 키스톤에 이미지 서비스 추가

[1] controller

source ~/keystone_admin

openstack user create --domain default --password-prompt placement

openstack role add --project service --user placement admin

openstack service create --name placement --description "Placement API" placement

openstack endpoint create --region RegionOne placement public http://controller:8778

openstack endpoint create --region RegionOne placement internal http://controller:8778

openstack endpoint create --region RegionOne placement admin http://controller:8778

​

​

(3) 서비스 설치

[1] controller

dnf -y install openstack-placement-api

​

(4) 서비스 설정

[1] controller

vi /etc/placement/placement.conf

위 파일 들어가서 아래 내용을 추가하는데

/[placement_database 로 해당하는 부분을 찾아서,

이미 있으면 수정하고 없는 부분은 제일 아래에 추가함.

​

[placement_database] connection = mysql+pymysql://placement:비번@controller/placement [api] auth_strategy = keystone [keystone_authtoken] auth_url = http://controller:5000/v3 memcached_servers = controller:11211 auth_type = password project_domain_name = Default user_domain_name = Default project_name = service username = placement password = 비번

su -s /bin/sh -c "placement-manage db sync" placement
