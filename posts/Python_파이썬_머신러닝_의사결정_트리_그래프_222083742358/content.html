3 ++. 주피터 환경이라면 위 과정으로도 아래와 같은 결과물이 나올 것이다.

그치만 파이참의 경우 재실행 후에도 그래프가 보이지 않으면 ~~

코드를 아래와 같이 수정한다.

​

from sklearn import tree clf = tree.DecisionTreeClassifier() from sklearn.datasets import load_iris ###################################### import graphviz dot_data = tree.export_graphviz(clf, out_file='tree.dot', feature_names=iris.feature_names, class_names=iris.target_names, filled=True, rounded=True, special_characters=True) from os import system system("dot -Tpng tree.dot -o tree.png")

​

두 코드에서 달라진 부분은 두 부분 인데,

1)

export_graphviz(clf, out_file=None, 이것을 -> export_graphviz(clf, out_file='tree.dot',

주피터 환경에서는 웹으로 바로 보이니까 따로 저장시킬 일이 없으니 None 이었지만,

파이참의 경우엔 tree.dot 이라는 파일을 생성하여 따로 파일로 저장시켜야 한다.

​

​

2)

graph = graphviz.Source(dot_data) graph 이것을 -> system("dot -Tpng tree.dot -o tree.png")

png 파일로 저장.
