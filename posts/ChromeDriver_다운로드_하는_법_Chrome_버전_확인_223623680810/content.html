크롤링 프로그램을 만들기 위해서는 ChromeDriver가 필요하다.

그 첫걸음으로 현재 사용 중인 Chrome 브라우저 버전에 맞는 버전의 ChromeDriver 설치해야 하는데ChromeDriver 는 본인의 Chrome 버전과 맞는 것으로 설치해야 하기 때문에 먼저 Chrome의 버전을 확인하는 것이 중요했다.

​

하지만, 이제 webdriver-manager를 사용하여 더 이상 ChromeDriver 버전을 일일이 찾아서 설치할 필요가 없다.

​

​

1. Chrome 버전 확인하기

먼저 Chrome 브라우저 버전을 확인하는 방법은 다음과 같다.

​

  1. 우측 상단 메뉴에서 설정 > Chrome 정보로 이동

​

​

2. 주소창에 chrome://settings/help를 입력해 바로 이동

Chrome 버전 확인 화면

​

이렇게 확인한 나의 Chrome 버전은 129.0.6668.103 이다.

​

​

​

ChromeDriver 설치 방법

이제 이 버전에 맞는 ChromeDriver를 설치해보자.

ChromeDriver 버전 선택 가이드를 참고해 사용 중인 Chrome의 주 버전(예: 129)과 맞는 ChromeDriver를 찾을 수 있다.

​

아하....

예전에는 여기서 내 버전과 맞는 것을 찾아

https://chromedriver.storage.googleapis.com/index.html 여기서 내 버전에 맞는 chromedriver 를 찾아서 다운받아 설치했었는데, 이제는 그럴 필요가 없다고 한다.

​

그냥 webdriver-manager 를 받으면, 알아서 내 버전에 맞는 chromedriver가 설치된다고.

​

​

2. webdriver-manager로 자동 ChromeDriver 설치

Python의 webdriver-manager 라이브러리를 사용하면, Python 프로그램이 자동으로 현재 Chrome 브라우저 버전에 맞는 ChromeDriver를 다운로드하고 관리해준다.

​

​

webdriver-manager 설치

아래 명령어를 사용해 webdriver-manager를 설치한다.

pip install webdriver-manager

​

설치가 완료됐고, 어떤 버전이 설치 됐는지 확인해보고자 간단한 버전 출력 프로그램을 만들어서 실행해보았다.

​

​

설치된 ChromeDriver 버전 확인

from webdriver_manager.chrome import ChromeDriverManager # ChromeDriver의 경로 확인 driver_path = ChromeDriverManager().install() print(f"Downloaded ChromeDriver path: {driver_path}")

​

잘 설치 됐네~

​
